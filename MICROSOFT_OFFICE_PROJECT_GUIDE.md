## üé® Design & Formatting Standards

### Color Palette Guidelines
- **Primary Colors:** 1-2 professional colors
- **Accent Color:** 1 complementary color
- **Background:** White or light neutral
- **Text:** Dark gray or black for readability

**Recommended Palettes:**

- **Spacing:** 1.5 line spacing in body text

## üÜò Troubleshooting & FAQ

**Q: My Word document looks different on another computer**
- A: Use standard fonts (Arial, Calibri, Times New Roman) and embed any custom fonts

**Q: Excel formulas are showing instead of results**
- A: Press Ctrl+` to toggle formula view, or right-click and select "Format Cells"

**Q: PowerPoint slides look different in presentation mode**
- A: Check slide master settings and test on multiple computers

**Q: Embedded Excel object in Word won't update**
- A: Right-click the object and select "Update Link" or re-embed the file

**Q: Hyperlinks aren't working**
- A: Ensure files are in the same folder or use absolute file paths
