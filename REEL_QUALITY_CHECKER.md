## SECTION 1: HOOK STRENGTH (40 points)

### Frame 1 Visual Quality (20 points)
- [ ] **20 pts** - Stunning/iconic landmark or unique perspective (will stop 80% of scrollers)
- [ ] **15 pts** - Very good photo, clear subject, professional quality
- [ ] **10 pts** - Good photo but not extraordinary (similar to 100 other travel accounts)
- [ ] **5 pts** - Mediocre/boring visual (sky, ocean, generic street)
- [ ] **0 pts** - Blurry, dark, or unclear subject

**Your Frame 1 score:** ___/20

### Frame 1 Text Hook (10 points)
- [ ] **10 pts** - Specific number + location ("$39 view", "300 steps", "UNESCO site")
- [ ] **7 pts** - Good hook but less specific ("Amazing view", "Hidden gem")
- [ ] **4 pts** - Generic text ("Check this out", "Beautiful place")
- [ ] **0 pts** - No text OR question format ("Should you visit?")

**Your text hook score:** ___/10

### Hook Timing (10 points)
- [ ] **10 pts** - Text appears in first 0.3 seconds + visual is strong immediately
- [ ] **7 pts** - Text appears within 0.5 seconds
- [ ] **4 pts** - Text delayed 1-2 seconds (people already skipping)
- [ ] **0 pts** - Text appears after 2+ seconds OR no text

**Your timing score:** ___/10

---

## SECTION 2: RETENTION STRENGTH (30 points)

### Frame Quality Consistency (15 points)
- [ ] **15 pts** - Every frame is visually strong (no weak/boring shots)
- [ ] **10 pts** - 2/3 frames are strong, 1 is mediocre
- [ ] **5 pts** - Only Frame 1 is strong, rest are filler
- [ ] **0 pts** - Multiple weak frames (ocean with ships, generic landscape)

**Your frame quality score:** ___/15

### Duration (10 points)
- [ ] **10 pts** - 5-6 seconds total (optimal for cultural travel)
- [ ] **7 pts** - 7-8 seconds (acceptable but risky)
- [ ] **4 pts** - 9-10 seconds (too long, skip rate will suffer)
- [ ] **0 pts** - 11+ seconds (guaranteed 75%+ skip rate)

**Your duration score:** ___/10

### Frame Variety (5 points)
- [ ] **5 pts** - Each frame shows different angle/perspective (wide → close-up → detail)
- [ ] **3 pts** - Some variety but frames feel similar
- [ ] **0 pts** - All frames look the same OR random/disconnected

**Your variety score:** ___/5

---

## SECTION 3: VALUE DELIVERY (20 points)

### Caption Quality (10 points)
- [ ] **10 pts** - Specific logistics (train line, price, hours, how to get there)
- [ ] **7 pts** - Some practical info but vague ("Go in the morning")
- [ ] **4 pts** - Personal story only, no actionable tips
- [ ] **0 pts** - Generic caption or just hashtags

**Your caption score:** ___/10

### Saveability (10 points)
*Would a stranger planning this trip save this for later?*

- [ ] **10 pts** - YES - Contains info they can't easily Google (entrance hack, timing tip, price comparison)
- [ ] **5 pts** - MAYBE - Has some value but not essential
- [ ] **0 pts** - NO - Just aesthetic/personal, no planning value

**Your saveability score:** ___/10

- [ ] **1 pts** - Somewhat on-brand but confusing (aesthetic sunset, lifestyle content)

## QUICK PRE-POST CHECKLIST

Before hitting "Post," ask yourself:

1. **Would I skip this in my feed?** (Be honest)
   - If YES → Don't post
   - If NO → Continue

2. **Is Frame 1 my BEST photo from this location?**
   - If NO → Replace it
   - If YES → Continue

3. **Does the text in Frame 1 have a specific number or unique fact?**
   - If NO → Rewrite text
   - If YES → Continue

4. **Is this 7 seconds or less?**
   - If NO → Cut frames
   - If YES → Continue

5. **Would a stranger save this to plan their trip?**
   - If NO → Add logistics to caption
   - If YES → Post it

**If you answer YES to all 5 → Post score is likely 75+**
