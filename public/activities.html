Activity 1: Temperature Converter

Easy

üéØ Goal

Create a function that converts temperatures between Fahrenheit and Celsius. The function should accept two parameters: the temperature value and the conversion type.

üìù Step-by-Step Instructions

  1. Define a function called convert_temperature(temp, to_celsius)
  2. Add a parameter to_celsius that is True when converting to Celsius, False when converting to Fahrenheit
  3. Use an if/else statement to check which conversion to perform
  4. Formula for F to C: (temp - 32) * 5/9
  5. Formula for C to F: (temp * 9/5) + 32
  6. Return the converted temperature rounded to 2 decimal places
  7. Test your function with: 100¬∞F ‚Üí C and 37¬∞C ‚Üí F

‚úÖ Expected Output

100¬∞F is 37.78¬∞C
37¬∞C is 98.6¬∞F

üí° Hints

  • Use the round() function to round to 2 decimal places
  • Remember: if statements check boolean conditions
  • The formula order matters - parentheses are important!

‚ú® Solution

def convert_temperature(temp, to_celsius):
    """
    Convert temperature between Fahrenheit and Celsius
    
    Parameters:
    temp (float): Temperature value to convert
    to_celsius (bool): True to convert F‚ÜíC, False to convert C‚ÜíF
    
    Returns:
    float: Converted temperature rounded to 2 decimal places
    """
    if to_celsius:
        # Fahrenheit to Celsius
        result = (temp - 32) * 5/9
    else:
        # Celsius to Fahrenheit
        result = (temp * 9/5) + 32
    
    return round(result, 2)

# Test the function
fahrenheit = 100
celsius = convert_temperature(fahrenheit, True)
print(f"{fahrenheit}¬∞F is {celsius}¬∞C")

celsius_temp = 37
fahrenheit = convert_temperature(celsius_temp, False)
print(f"{celsius_temp}¬∞C is {fahrenheit}¬∞F")

üìñ Explanation

Function Definition: We define a function with two parameters - the temperature and a boolean flag.

Conditional Logic: The if statement checks which conversion to perform based on the to_celsius parameter.

Mathematical Operations: Each branch applies the correct conversion formula.

Return Value: We use round() to limit the result to 2 decimal places for readability.

üîë Key Concepts
  • Function parameters and return values
  • Boolean parameters for controlling behavior
  • If/else conditional statements
  • Mathematical operations in Python
  • String formatting with f-strings

Activity 2: Grade Calculator

Medium

üéØ Goal

Create a function that calculates the average grade from a list of scores and assigns a letter grade. Use a for loop to sum the scores and if/elif/else to determine the letter grade.

üìù Step-by-Step Instructions

  1. Define a function called calculate_grade(scores) that takes a list of numbers
  2. Initialize a variable total = 0 to store the sum
  3. Use a for loop to iterate through each score in the list
  4. Add each score to the total inside the loop
  5. Calculate average: total / len(scores)
  6. Use if/elif/else to assign letter grades:
    • A: 90-100
    • B: 80-89
    • C: 70-79
    • D: 60-69
    • F: below 60
  7. Return both the average and letter grade

‚úÖ Expected Output

Scores: [85, 92, 78, 90, 88]
Average: 86.6
Letter Grade: B

Scores: [95, 98, 93, 97, 100]
Average: 96.6
Letter Grade: A

üí° Hints

  • Use for score in scores: to iterate
  • Add to total with total += score
  • Use len(scores) to get the count
  • Use >= for grade boundaries
  • Round average to 1 decimal place

‚ú® Solution

def calculate_grade(scores):
    """
    Calculate average and letter grade from a list of scores
    
    Parameters:
    scores (list): List of numerical scores
    
    Returns:
    tuple: (average, letter_grade)
    """
    # Calculate sum using for loop
    total = 0
    for score in scores:
        total += score
    
    # Calculate average
    average = total / len(scores)
    
    # Determine letter grade using if/elif/else
    if average >= 90:
        letter = 'A'
    elif average >= 80:
        letter = 'B'
    elif average >= 70:
        letter = 'C'
    elif average >= 60:
        letter = 'D'
    else:
        letter = 'F'
    
    return round(average, 1), letter

# Test the function
test_scores_1 = [85, 92, 78, 90, 88]
avg1, grade1 = calculate_grade(test_scores_1)
print(f"Scores: {test_scores_1}")
print(f"Average: {avg1}")
print(f"Letter Grade: {grade1}")
print()

test_scores_2 = [95, 98, 93, 97, 100]
avg2, grade2 = calculate_grade(test_scores_2)
print(f"Scores: {test_scores_2}")
print(f"Average: {avg2}")
print(f"Letter Grade: {grade2}")

üìñ Explanation

For Loop: We iterate through each score in the list, adding it to our running total.

Average Calculation: Divide the total by the count of scores using len().

If/Elif/Else Chain: We check grade boundaries from highest to lowest. Once a condition is met, the rest are skipped.

Multiple Return Values: Python functions can return multiple values as a tuple.

üîë Key Concepts
  • For loops for iterating over lists
  • Accumulator pattern (building up a total)
  • If/elif/else chains for multiple conditions
  • Returning multiple values (tuples)
  • Comparison operators (>=, <)

Activity 3: Password Validator

Medium

üéØ Goal

Create a function that validates if a password meets security requirements. Use a while loop to give the user 3 attempts to enter a valid password.

üìù Step-by-Step Instructions

  1. Define a function called is_valid_password(password) that checks:
    • Length is at least 8 characters
    • Contains at least one digit
    • Contains at least one uppercase letter
  2. Return True if all conditions are met, False otherwise
  3. Create another function password_prompt()
  4. Initialize attempts = 0 and max_attempts = 3
  5. Use a while loop that runs while attempts < max_attempts
  6. Inside the loop:
    • Ask for password input
    • Check if valid
    • If valid, return success message and break
    • If not, increment attempts and show error
  7. After loop: Show "locked out" message if all attempts failed

‚úÖ Expected Output

Enter password: hello
‚ùå Invalid! Must be 8+ chars, have digit, have uppercase
Attempts remaining: 2

Enter password: Hello123
‚úÖ Password accepted! Access granted.

üí° Hints

  • Use len(password) >= 8 for length
  • Use any(c.isdigit() for c in password) to check for digits
  • Use any(c.isupper() for c in password) to check for uppercase
  • Use and to combine all conditions
  • Remember to increment attempts!

‚ú® Solution

def is_valid_password(password):
    """
    Check if password meets security requirements
    
    Requirements:
    - At least 8 characters long
    - Contains at least one digit
    - Contains at least one uppercase letter
    
    Parameters:
    password (str): Password to validate
    
    Returns:
    bool: True if valid, False otherwise
    """
    # Check length
    has_length = len(password) >= 8
    
    # Check for at least one digit
    has_digit = any(c.isdigit() for c in password)
    
    # Check for at least one uppercase letter
    has_upper = any(c.isupper() for c in password)
    
    # All conditions must be True
    return has_length and has_digit and has_upper

def password_prompt():
    """
    Prompt user for valid password with 3 attempts
    """
    attempts = 0
    max_attempts = 3
    
    while attempts < max_attempts:
        password = input("Enter password: ")
        
        if is_valid_password(password):
            print("‚úÖ Password accepted! Access granted.")
            return True
        else:
            attempts += 1
            remaining = max_attempts - attempts
            print("‚ùå Invalid! Must be 8+ chars, have digit, have uppercase")
            
            if remaining > 0:
                print(f"Attempts remaining: {remaining}")
                print()
    
    print("üîí Account locked! Too many failed attempts.")
    return False

# Test the function

# For demonstration without input:

print(f"'Hello123' is valid: {is_valid_password('Hello123')}")
print(f"'Pass' is valid: {is_valid_password('Pass')}")
print(f"'Password1' is valid: {is_valid_password('Password1')}")

üìñ Explanation

Validation Function: We break down password requirements into separate boolean checks, then combine them with and.

While Loop: Continues running as long as attempts haven't reached the maximum. This is perfect for retry logic.

Early Exit: When a valid password is entered, we return immediately, breaking out of the loop.

Counter Pattern: We increment attempts each iteration to track progress toward the limit.

üîë Key Concepts
  • While loops with conditional logic
  • String methods (isdigit(), isupper())
  • Generator expressions with any()
  • Boolean logic and compound conditions
  • Loop control with return/break
  • Counter pattern for tracking attempts

Activity 4: Number Guessing Game

Hard

üéØ Goal

Create a complete number guessing game that combines functions, while loops, and if/elif/else statements. The computer picks a random number and the player has to guess it with hints.

üìù Step-by-Step Instructions

  1. Import random module at the top of your file
  2. Define function play_guessing_game(min_num, max_num, max_guesses)
  3. Generate random number: secret = random.randint(min_num, max_num)
  4. Initialize guesses = 0
  5. Use while loop that continues while guesses < max_guesses
  6. Inside loop:
    • Get player's guess (convert to int)
    • Increment guesses counter
    • Use if/elif/else to check:

      • If too high ‚Üí give "Too high!" hint
      • If too low ‚Üí give "Too low!" hint

  7. After loop: If not guessed, reveal answer and return False

‚úÖ Expected Output

‚ú® Solution

import random

def play_guessing_game(min_num=1, max_num=100, max_guesses=7):
    """
    Play a number guessing game
    
    Parameters:
    min_num (int): Minimum number in range
    max_num (int): Maximum number in range
    max_guesses (int): Maximum number of guesses allowed
    
    Returns:
    bool: True if player won, False if ran out of guesses
    """
    # Generate random secret number
    secret = random.randint(min_num, max_num)
    guesses = 0
    
    print(f"üé≤ Guess the number between {min_num} and {max_num}!")
    print(f"You have {max_guesses} guesses.\n")
    
    # Main game loop
    while guesses < max_guesses:
        try:
            # Get player's guess
            guess = int(input(f"Guess #{guesses + 1}: "))
            guesses += 1
            
            # Check the guess
            if guess == secret:
                print(f"üéâ Correct! You won in {guesses} guesses!")
                return True
            elif guess > secret:
                remaining = max_guesses - guesses
                print("Too high! Try again.")
                if remaining > 0:
                    print(f"Remaining guesses: {remaining}\n")
            else:  # guess < secret
                remaining = max_guesses - guesses
                print("Too low! Try again.")
                if remaining > 0:
                    print(f"Remaining guesses: {remaining}\n")
        
        except ValueError:
            print("‚ùå Invalid input! Please enter a number.\n")
            # Don't count invalid input as a guess
            guesses -= 1
    
    # Out of guesses
    print(f"\nüíî Game Over! The number was {secret}.")
    return False

# Play the game

# play_guessing_game()

# For demonstration:
print("Number Guessing Game - Ready to play!")
print("Uncomment the play_guessing_game() line to start.")

üìñ Explanation

Random Number Generation: We use random.randint() to pick a number in the specified range.

While Loop with Limit: The game continues until max guesses are used or the player wins.

Nested If/Elif/Else: We check three cases: correct guess, too high, or too low.

Try/Except Block: Handles invalid input (non-numbers) gracefully without crashing.

Return Values: Function returns True/False to indicate win/loss, which could be used for tracking statistics.

üîë Key Concepts
  • Importing and using modules (random)
  • Function with default parameters
  • While loop with multiple exit conditions
  • If/elif/else for game logic
  • Exception handling (try/except)
  • User input and type conversion
  • Boolean return values

def custom_fizzbuzz(n, rules):
    """
    Enhanced FizzBuzz with custom rules
    
    Parameters:
    n (int): Count from 1 to n
    rules (dict): Dictionary mapping divisor to replacement word
                  Example: {3: "Fizz", 5: "Buzz"}
    
    Returns:
    int: Count of special numbers (non-numeric outputs)
    """
    special_count = 0
    
    # Process each number from 1 to n
    for num in range(1, n + 1):
        output = ""
        
        # Check each rule
        for divisor, word in rules.items():
            if num % divisor == 0:
                output += word
        
        # If no rules matched, use the number
        if not output:
            output = str(num)
        else:
            special_count += 1
        
        print(output)
    
    return special_count

üìñ Explanation

Flexible Design: By accepting a rules dictionary, this function can handle any FizzBuzz-style problem.

Nested Loops: The outer for loop processes numbers, the inner loop checks each rule.

String Building: We concatenate words when multiple rules match (e.g., 15 ‚Üí "FizzBuzz").

Conditional Output: If no rules matched, we output the number; otherwise, we output the built string.

Tracking Statistics: We count and return how many special numbers were found.
