The Bug Challenge
For a score of 95, the correct grade should be 'A'. However, the current code outputs 'C' due to improper arrangement of conditional statements.
Buggy Code (Copy and Fix This) 
# Bug 1: Incorrect Elif Order
score = 95
if score >= 70:
    grade = 'C'
elif score >= 80:
    grade = 'B'
elif score >= 90:
    grade = 'A'
print("Grade:", grade)

Problem Analysis

Expected Output: The grade should be 'A' for a score of 95.

Actual Output: The code outputs 'C' instead.

Why this happens: The conditions are checked from top to bottom. Since 95 >= 70 is true, the program assigns 'C' and never checks the other conditions.

Key Learning Points:

  • Order matters: In if-elif-else statements, conditions are checked from top to bottom
  • Start with the most restrictive condition: Begin with the highest score requirements
  • Use else for default cases: Handle scores below 70 with an else statement
  • Test edge cases: Always test your code with boundary values like 90, 80, 70
