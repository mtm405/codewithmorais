The Bug Challenge
The code must correctly identify the year 2024 as a leap year, but there's a syntax error preventing it from running.
Buggy Code (Copy and Fix This) 
# Bug 2: Assignment vs. Comparison
year = 2024
is_leap = False
if year % 4 = 0:  # Syntax Error Here!
    is_leap = True
print("Is", year, "a leap year?", is_leap)

Problem Analysis

Expected Output: The code should run and print "Is 2024 a leap year? True"

Actual Output: SyntaxError: invalid syntax

Why this happens: The code uses assignment operator (=) instead of comparison operator (==) in the if statement.

Key Learning Points:

  • Assignment vs Comparison: = assigns a value, == compares values
  • Syntax Error: Python can't parse invalid syntax like using = in conditions
  • Operator Precedence: Comparison operators are used in conditional statements
  • Debugging Tip: Look for = vs == errors in conditional statements

Hint

In Python, = is used to assign values to variables, while == is used to compare values. Check your if statement!
