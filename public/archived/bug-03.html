The Bug Challenge
The code must correctly identify the number 5 as "Outside" the range of 10-20, but it's using the wrong logical operator.
Buggy Code (Copy and Fix This) 
# Bug 3: Wrong Logical Operator
num = 5
if num < 10 and num > 20:  # Wrong logical operator!
    result = "Outside"
else:
    result = "Inside"
print("Number", num, "is:", result)

Problem Analysis

Expected Output: Number 5 is: Outside

Actual Output: Number 5 is: Inside

Why this happens: The condition uses "and" but should use "or". A number can't be both less than 10 AND greater than 20 at the same time.

Key Learning Points:

  • Logical AND: Both conditions must be true for the whole expression to be true
  • Logical OR: Only one condition needs to be true for the whole expression to be true
  • Range Logic: To check if outside a range, use OR (less than min OR greater than max)
  • Common Mistake: Confusing AND/OR in range checking logic
