Problem Analysis

Expected Output: Success

Actual Output: SyntaxError: invalid syntax

Why this happens: Python requires a colon (:) at the end of if statements to indicate the start of the code block.

Key Learning Points:

  • Colon Requirement: Python requires colons after if, elif, else, while, for, def, and class statements
  • Code Block Syntax: The colon indicates where the code block begins
  • Indentation: After the colon, the next lines must be indented to belong to the block
  • Common Error: Forgetting colons is one of the most common Python syntax errors
