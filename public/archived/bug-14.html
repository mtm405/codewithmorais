Buggy Code

animals = ["cat", "dog", "bird"]

print("My favorite animals:")

    print(f"- {animal}")

print("Those are all my pets!")

Debugging Tips

  • len() returns a number: len(animals) returns 3, not the list items
  • You can't iterate over a number: This will cause a TypeError
  • Direct iteration: Use 'for animal in animals' to get each item
  • List vs length: Understand the difference between the list and its length

What You Should Learn

  • The difference between iterating over a list and its length
  • How len() function returns an integer, not an iterable
  • Direct iteration over lists vs index-based iteration
  • When to use 'for item in list' vs 'for i in range(len(list))'
