Buggy Code

value = -5

print(f"Analyzing value: {value}")

if value >= 0:
    print("Non-negative")
elif value > 0:
    print("Positive")
elif value < 0:  # Bug: this will never be reached
    print("Negative")
else:
    print("This should never print")

print("Analysis complete!")

Debugging Tips

  • Condition order matters: Once a condition is true, the rest are skipped
  • Overlapping conditions: value >= 0 includes value > 0, making the second condition unreachable
  • Logical flow: Think about which conditions should be checked first
  • Test all paths: Make sure every branch can be reached with some input
