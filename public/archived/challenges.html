Simple Calculator
Beginner
Create a calculator that asks the user for two numbers and an operation (+, -, *, /) and displays the result.

Requirements:

  • Ask for two numbers
  • Ask for an operation
  • Display the result
Example Run:
Enter first number: 10
Enter second number: 5
Enter operation (+, -, *, /): +
Result: 15 
num1 = float(input("Enter first number: "))
num2 = float(input("Enter second number: "))
operation = input("Enter operation (+, -, *, /): ")

if operation == "+":
    result = num1 + num2
elif operation == "-":
    result = num1 - num2
elif operation == "*":
    result = num1 * num2
elif operation == "/":
    result = num1 / num2
else:
    result = "Invalid operation"

print(f"Result: {result}")

Prime Number Checker
Advanced
Write a function that checks if a given number is prime (only divisible by 1 and itself).

Requirements:

  • Create a function is_prime()
  • Ask user for a number
  • Return True if prime, False if not
  • Display the result clearly
ðŸ’¡ Hint:
Check divisibility from 2 to the square root of the number
def is_prime(n):
    if n < 2:
        return False
    if n == 2:
        return True
    if n % 2 == 0:
        return False
    
    for i in range(3, int(n**0.5) + 1, 2):
        if n % i == 0:
            return False
    return True

number = int(input("Enter a number: "))

if is_prime(number):
    print(f"{number} is a prime number!")
else:
    print(f"{number} is not a prime number.")

Fibonacci Sequence
Advanced
Generate the Fibonacci sequence up to n terms, where each number is the sum of the two preceding ones.

Requirements:

  • Ask user how many terms to generate
  • Generate the Fibonacci sequence
  • Display all numbers in the sequence
  • Handle edge cases (n â‰¤ 0)
Example (n=7):
Fibonacci sequence: 0, 1, 1, 2, 3, 5, 8 
n = int(input("How many terms? "))

if n <= 0:
    print("Please enter a positive number")
elif n == 1:
    print("Fibonacci sequence: 0")
else:
    fibonacci = [0, 1]
    
    for i in range(2, n):
        next_num = fibonacci[i-1] + fibonacci[i-2]
        fibonacci.append(next_num)
    
    print("Fibonacci sequence:", ", ".join(map(str, fibonacci)))

Rock Paper Scissors
Intermediate
Create a Rock, Paper, Scissors game where the user plays against the computer.

Requirements:

  • Ask user for their choice (rock, paper, scissors)
  • Generate random computer choice
  • Determine and display the winner
  • Show both choices
import random

choices = ["rock", "paper", "scissors"]
computer_choice = random.choice(choices)
user_choice = input("Enter rock, paper, or scissors: ").lower()

print(f"You chose: {user_choice}")
print(f"Computer chose: {computer_choice}")

if user_choice == computer_choice:
    print("It's a tie!")
elif (user_choice == "rock" and computer_choice == "scissors") or \
     (user_choice == "paper" and computer_choice == "rock") or \
     (user_choice == "scissors" and computer_choice == "paper"):
    print("You win!")
else:
    print("Computer wins!")

Shopping List Manager
