What is Control Flow?

Control flow determines the order in which code is executed. It allows programs to make decisions and repeat actions.

If Statements:

if condition:
    # code to execute if true
elif another_condition:
    # code for another condition
else:
    # code if all conditions are false

For Loops:

for item in range(5):
    print(item)  # prints 0, 1, 2, 3, 4

While Loops:

count = 0
while count < 5:
    print(count)
    count += 1

Important Notes:

  • Python uses indentation to group code blocks
  • Use 4 spaces for each indentation level
  • Conditions must evaluate to True or False
