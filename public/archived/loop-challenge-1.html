Challenge 1: The Dynamic Mascot Picker

Challenge Description

Goal: Continuously pick mascots for two rival schools until the user types "stop" or the two randomly chosen mascots happen to be identical (a rare event), which then stops the picking process immediately with a special message.

How to Do It:

  1. Import the random module
  2. Create two lists of mascots, ensuring at least one mascot is common to both lists
  3. Start a while True loop
  4. Inside the loop, use random.choice() to pick one mascot from each list
  5. Ask the user for input (e.g., "Press Enter to pick or type 'stop' to quit:")
  6. Use an if statement to check if the user input is "stop" and use break to exit
  7. Use a second if statement to check if the two chosen mascots are identical. If they are, print the special message and use break to exit
  8. If neither condition is met, print the chosen, non-identical mascots and let the loop continue
üí° Tip: Make sure to include at least one common mascot in both lists (like "Eagles" or "Tigers") so there's a chance for the matching condition to occur!

Sample Solution:

import random # Create mascot lists with at least one common mascot school_a_mascots = ["Lions", "Tigers", "Eagles", "Bulldogs", "Panthers"] school_b_mascots = ["Bears", "Tigers", "Hawks", "Wildcats", "Eagles"] while True: # Pick mascots for both schools mascot_a = random.choice(school_a_mascots) mascot_b = random.choice(school_b_mascots) print(f"School A picked: {mascot_a}") print(f"School B picked: {mascot_b}") # Get user input user_input = input("Press Enter to pick or type 'stop' to quit: ") # Check if user wants to stop if user_input.lower() == "stop": print("Thanks for playing!") break # Check if mascots match if mascot_a == mascot_b: print("üéâ MATCH FOUND! Both schools picked " + mascot_a + "!") print("The mascot picking stops here due to this rare coincidence!") break print() # Empty line for readability
