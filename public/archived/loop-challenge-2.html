Challenge 2: Required Study Topics

Basic Level

Challenge Description

Goal: A student has a list of 5 topics they must study. Use a for loop to cycle through the list. If a topic is "Calculus" or "Physics," use continue to print a message like "Skipping Calculus for now, it needs a separate session." and move immediately to the next topic.

How to Do It:

  1. Create a single list of five diverse topics, including "Calculus" and "Physics"
  2. Use a for topic in list: loop to iterate through the topics
  3. Inside the loop, use an if topic == "..." condition to check for "Calculus." Print the skip message and execute continue
  4. Use an elif condition to do the same for "Physics"
  5. After the if/elif block, use a print() statement to suggest studying the current topic (this line will be skipped if continue is executed)
üí° Key Concept: The continue statement skips the rest of the current iteration and moves immediately to the next item in the loop!

Expected Output

Skipping Calculus for now, it needs a separate session. Let's study History today! Skipping Physics for now, it needs a separate session. Let's study English today! Let's study Biology today!

Sample Solution:

# List of study topics study_topics = ["Calculus", "History", "Physics", "English", "Biology"] # Loop through each topic for topic in study_topics: # Check if topic is Calculus if topic == "Calculus": print("Skipping Calculus for now, it needs a separate session.") continue # Check if topic is Physics elif topic == "Physics": print("Skipping Physics for now, it needs a separate session.") continue # This line only executes for non-skipped topics print(f"Let's study {topic} today!")
üîç What to Look For: Your program should print skip messages for Calculus and Physics, but study messages for all other topics. The continue statements prevent the final print from executing for the skipped subjects.
