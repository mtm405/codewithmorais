Challenge 3: The "Would You Rather" Gauntlet

Basic Level
Back to All Challenges Intermediate Advanced Home

Challenge Description

Goal: Present the user with a sequence of "Would You Rather" questions. After each question, ask the user if they want to keep playing. The game continues indefinitely until the user inputs "no," which triggers a break to end the game.

How to Do It:

  1. Import the random module
  2. Create a list of "Would You Rather" questions
  3. Start a while True loop
  4. Inside the loop, use random.choice() to pick and print a question
  5. Use input() to ask the user if they want to keep playing (e.g., "Keep playing? (yes/no): ")
  6. Use an if statement to check if the user input (in lowercase) is equal to "no." If it is, print a goodbye message and execute break
üí° Key Concept: Use .lower() on the input to handle both "No" and "no" responses!

Expected Output

Would you rather have the ability to fly or be invisible? Keep playing? (yes/no): yes Would you rather always be 10 minutes late or always be 20 minutes early? Keep playing? (yes/no): yes Would you rather have unlimited pizza for life or unlimited ice cream for life? Keep playing? (yes/no): no Thanks for playing the Would You Rather game! üéÆ

Sample Solution:

import random # List of "Would You Rather" questions questions = [ "Would you rather have the ability to fly or be invisible?", "Would you rather always be 10 minutes late or always be 20 minutes early?", "Would you rather have unlimited pizza for life or unlimited ice cream for life?", "Would you rather never have to sleep or never have to eat?", "Would you rather be able to speak all languages or talk to animals?", "Would you rather live in the past or the future?", "Would you rather have super strength or super speed?" ] while True: # Pick and display a random question question = random.choice(questions) print(question) # Ask if user wants to continue user_choice = input("Keep playing? (yes/no): ") # Check if user wants to quit if user_choice.lower() == "no": print("Thanks for playing the Would You Rather game! üéÆ") break print() # Empty line for readability
üîç What to Look For: Your program should randomly select questions, handle user input, and properly exit when the user types "no". The game continues until the user decides to quit.
