Challenge 4: Lunch Decision Loop

Challenge Description

Goal: Ask the user if they want a "hot" or "cold" lunch. If they give an invalid input (e.g., "salad"), keep asking (looping) until they enter a valid choice ("hot" or "cold") or type "none."

How to Do It:

  1. Import the random module
  2. Create lists for hot_lunch and cold_lunch
  3. Start a while True loop
  4. Get the user's input
  5. Use an if statement to check if the input is "none." If it is, print a message and break
  6. Use an elif to check for "hot." If true, pick and print a hot lunch, then break
  7. Use another elif to check for "cold." If true, pick and print a cold lunch, then break
  8. Use a final else block to handle invalid input. Print an error message, and the loop will automatically continue
üí° Key Concept: Input validation using loops! The program keeps asking until it gets a valid response.

Please choose 'hot', 'cold', or 'none'. Do you want a hot or cold lunch? (hot/cold/none): hot Here's your hot lunch: Grilled Chicken Sandwich! üî•

Sample Solution:

import random # Create lunch options hot_lunch = ["Grilled Chicken Sandwich", "Beef Stew", "Hot Pizza Slice", "Soup and Bread", "Pasta with Marinara"] cold_lunch = ["Caesar Salad", "Turkey Club Sandwich", "Fruit Bowl", "Yogurt Parfait", "Cold Cut Wrap"] while True: # Get user's preference choice = input("Do you want a hot or cold lunch? (hot/cold/none): ").lower() # Check if user doesn't want lunch if choice == "none": print("No lunch today. Have a great day! üòä") break # Check for hot lunch elif choice == "hot": selected_lunch = random.choice(hot_lunch) print(f"Here's your hot lunch: {selected_lunch}! üî•") break # Check for cold lunch elif choice == "cold": selected_lunch = random.choice(cold_lunch) print(f"Here's your cold lunch: {selected_lunch}! üßä") break # Handle invalid input else: print("Invalid choice! Please choose 'hot', 'cold', or 'none'.") print() # Empty line for readability
