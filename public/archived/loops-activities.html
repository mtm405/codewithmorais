1. The Countdown Creator üöÄ

Goal: Create a simple countdown timer using a while loop. This activity reinforces the concepts of a loop condition and a counter variable.
How to Do It:
  1. Ask the user for a number to count down from using input(). Make sure to convert the input to an integer.
  2. Create a while loop that continues as long as the number is greater than 0.
  3. Inside the loop, use print() to display the current number.
  4. Inside the loop, subtract 1 from the number on each iteration. This is a crucial step to prevent an infinite loop.
  5. After the loop finishes, use print() to display a message like "Blastoff!".
# The Countdown Creator
countdown_number = int(input("Enter a number to count down from: "))
while countdown_number > 0:
    print(countdown_number)
    countdown_number -= 1
print("Blastoff!")

3. The Number Guessing Game üîÆ

Goal: Build a simple guessing game where the user keeps guessing a number until they get it right. This activity combines a while loop with user input and conditional statements.
How to Do It:
  1. Use import random at the beginning of your script.
  2. Create a variable secret_number and assign a random integer between 1 and 10 to it using random.randint().
  3. Create a while True loop to make the program repeat indefinitely.
  4. Inside the loop, use input() to ask the user to guess the number.
  5. Use an if/elif/else block to check the user's guess:
    • If the guess is equal to the secret_number, print a success message and use the break statement to exit the loop.
    • If the guess is too high, print "Too high! Try again."
    • If the guess is too low, print "Too low! Try again."
# The Number Guessing Game
import random
secret_number = random.randint(1, 10)
print("I'm thinking of a number between 1 and 10.")
while True:
    try:
        guess = int(input("What is your guess? "))
        if guess == secret_number:
            print("You got it! You win!")
            break  # This stops the loop
        elif guess > secret_number:
            print("Too high! Try again.")
        else:
            print("Too low! Try again.")
    except ValueError:
        print("Please enter a valid number.")

4. The Vowel Counter üó£Ô∏è

Goal: Count how many vowels are in a user-provided word. This activity challenges students to combine a for loop with a counter and conditional logic.
How to Do It:
  1. Create a variable vowel_count and initialize it to 0.
  2. Ask the user to enter a word using input().
  3. Create a for loop to iterate through each character in the word.
  4. Inside the loop, use a conditional statement to check if the current character is a vowel (e.g., if character == "a" or character == "e" ...).
  5. If the character is a vowel, increment vowel_count by 1.
  6. After the loop has finished, use print() to display the total number of vowels found in the word.
# The Vowel Counter
vowel_count = 0
word = input("Enter a word: ").lower()  # .lower() makes the check case-insensitive
for character in word:
    if character in "aeiou":
        vowel_count += 1
print(f"There are {vowel_count} vowels in the word.")

5. The "I'm Thinking Of..." Game üß†

Goal: This activity builds on the number guessing game by introducing a limited number of guesses using a while loop with a counter.
How to Do It:
  1. Import the random module.
  2. Choose a random secret_number between 1 and 10.
  3. Create a variable guesses_left and set it to 3.
  4. Start a while loop that runs as long as guesses_left > 0.
  5. Inside the loop, get the user's guess.
  6. Use an if/elif/else block to check the guess and provide feedback.
  7. Crucially, if the guess is wrong, subtract 1 from guesses_left.
  8. If the guess is correct, print a win message and use break.
  9. After the loop, use an else block or a final if statement to check if guesses_left is 0. If it is, print a "You ran out of guesses" message.
# The "I'm Thinking Of..." Game
import random
secret_number = random.randint(1, 10)
guesses_left = 3
print("I'm thinking of a number between 1 and 10. You have 3 tries!")
while guesses_left > 0:
    guess = int(input(f"You have {guesses_left} guesses left. What is your guess? "))
    if guess == secret_number:
        print("You got it! You win!")
        break
    elif guess > secret_number:
        print("Too high!")
    else:
        print("Too low!")
    guesses_left -= 1
if guesses_left == 0:
    print(f"You ran out of guesses! The number was {secret_number}.")

6. The "I Can Count!" Robot ü§ñ

Goal: Use a for loop with the range() function to make the program count in different ways.
How to Do It:
  1. Use a for loop with range(1, 11) to make the robot count from 1 to 10.
  2. Modify the range() function to count from 5 to 10.
  3. Modify the range() function again to count from 1 to 10, but this time only print the even numbers (by adding a third argument to range()).
# The "I Can Count!" Robot
# Part 1: Count from 1 to 10
print("Counting from 1 to 10:")
for i in range(1, 11):
    print(i)

# Part 2: Count from 5 to 10
print("\nCounting from 5 to 10:")
for i in range(5, 11):
    print(i)

# Part 3: Count even numbers
print("\nCounting even numbers from 1 to 10:")
for i in range(2, 11, 2):
    print(i)

7. The List Organizer üìö

Goal: Go through a list and create a new list containing only the items that meet a certain condition. This combines loops and list manipulation.
How to Do It:
  1. Create a list of words, with some short and some long words.
    words = ["cat", "elephant", "dog", "ant", "giraffe"]
  2. Create a new, empty list called long_words.
  3. Use a for loop to iterate through each word in the original words list.
  4. Inside the loop, use an if statement to check if the length of the current word is greater than 5 using len().
  5. If it is, use the .append() method to add that word to the long_words list.
  6. After the loop, print the long_words list to show the result.
# The List Organizer
words = ["cat", "elephant", "dog", "ant", "giraffe"]
long_words = []
for word in words:
    if len(word) > 5:
        long_words.append(word)
print(long_words)

8. The "No Skip" Challenge üö´

Goal: Use the continue statement to skip specific items in a loop.
How to Do It:
  1. Create a list of numbers from 1 to 10.
  2. Start a for loop to iterate through each number.
  3. Inside the loop, use an if statement to check if the current number is 5.
  4. If the number is 5, use the continue statement to skip the rest of the code in that iteration.
  5. After the if block, print() the current number.
  6. The result should be the numbers 1 through 10, but with the number 5 missing.
# The "No Skip" Challenge
for number in range(1, 11):
    if number == 5:
        continue
    print(number)

9. The Grid Maker üî≥

Goal: Create a simple grid using nested loops. This is a great way to introduce a more complex application of loops.
How to Do It:
  1. Use a for loop with range(3) to create the rows.
  2. Inside this loop, create another for loop with range(3) to create the columns.
  3. Inside the inner loop, print " O " with end="". The end="" prevents a new line after each "O", so they all print on the same line.
  4. After the inner loop, use print() with no arguments to start a new line for the next row.
  5. The result should be a 3x3 grid of "O"s.
# The Grid Maker
for i in range(3):
    for j in range(3):
        print(" O ", end="")
    print()  # This prints a new line

10. The while Loop with a "Secret Exit" üö™

Goal: Combine a while loop with a break statement to create a simple text-based program that can be exited by typing a specific word.
How to Do It:
  1. Create a while True loop, explaining that it will run forever unless a special command is given.
  2. Inside the loop, ask the user to type a message or "exit" using input().
  3. Use an if statement to check if the user's message is exactly "exit".
  4. If it is, print a "Goodbye!" message and use break to stop the loop.
  5. If the message is not "exit", print "You said: " followed by the message.
# The "Secret Exit"
while True:
    message = input("Type a message or type 'exit' to quit: ")
    if message == "exit":
        print("Goodbye!")
        break
    else:
        print("You said: " + message)
