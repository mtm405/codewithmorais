üßÆ Addition Calculator

Learn the difference between print() and return by creating a calculator function

Beginner

üéØ Your Mission

Create a function called add_numbers that takes two numbers and returns their sum (not prints it). This introduces the important concept of returning values from functions.

üìã Step-by-Step Instructions:

  1. Define the function: Use def add_numbers(num1, num2):
  2. Add two parameters: num1 and num2 will hold the numbers to add
  3. Calculate the sum: Add the two parameters together
  4. Return the result: Use return instead of print()
  5. Store the result: Save the function's output in a variable
  6. Display the result: Print the stored result with a message

üîë Key Concept: Return vs Print

return gives back a value you can use later. print() just shows something on screen. Functions that return values are more useful because you can store and reuse their results!

üîß Template to Get You Started:

def add_numbers(num1, num2): # Calculate the sum result = num1 + num2 # Return the result (don't print it) return result # Test your function by storing the results answer1 = add_numbers(5, 3) answer2 = add_numbers(10, 15) # Now print the results with nice messages print(f"5 + 3 = {answer1}") print(f"10 + 15 = {answer2}")

‚úÖ Expected Output:

5 + 3 = 8 10 + 15 = 25
üí° Understanding Return:
  • return num1 + num2 - sends the sum back to whoever called the function
  • result = add_numbers(5, 3) - stores the returned value in a variable
  • Without return, the function would give back None
Quick Test: Try print(add_numbers(7, 2)) - this should show 9!

üîç Complete Solution with Explanation:

def add_numbers(num1, num2): """ This function adds two numbers and returns the result. Parameters: num1 (number), num2 (number) Returns: the sum of num1 and num2 """ return num1 + num2 # Test the function by storing results in variables result1 = add_numbers(5, 3) # result1 now contains 8 result2 = add_numbers(10, 15) # result2 now contains 25 # Display the results with formatted messages print(f"5 + 3 = {result1}") print(f"10 + 15 = {result2}") # You can also use the function directly in print: print(f"7 + 2 = {add_numbers(7, 2)}")
üéì What You Learned:
  • The difference between return and print()
  • How to use multiple parameters in functions
  • How to store function results in variables
  • How to use returned values in other operations

üéÇ Age Calculator

Practice mathematical operations in functions by calculating someone's age

Beginner

üéØ Your Mission

Create a function called calculate_age that takes someone's birth year and the current year, then calculates and returns their age. This teaches you how to do math inside functions!

üìã Step-by-Step Instructions:

  1. Define the function: Use def calculate_age(birth_year, current_year):
  2. Understand the parameters: birth_year is when they were born, current_year is this year
  3. Do the math: Age = current year minus birth year
  4. Return the result: Give back the calculated age
  5. Test with examples: Try different birth years to make sure it works
  6. Display results: Print the ages with descriptive messages

üßÆ Math in Programming

Programming is great for automating calculations! The formula for age is simple: Age = Current Year - Birth Year. Your function will do this calculation automatically for any birth year.

üîß Template to Get You Started:

def calculate_age(birth_year, current_year): # Calculate the person's age age = current_year - birth_year # Return the calculated age return age # Test with different birth years age_2005 = calculate_age(2005, 2024) age_2010 = calculate_age(2010, 2024) # Display the results print(f"Born in 2005: {age_2005} years old") print(f"Born in 2010: {age_2010} years old")

‚úÖ Expected Output:

Born in 2005: 19 years old Born in 2010: 14 years old
üí° Mathematical Thinking:
  • If current year is 2024 and birth year is 2005, then 2024 - 2005 = 19
  • You can do the calculation in one line: return current_year - birth_year
  • Or store it first: age = current_year - birth_year, then return age
Try This: What happens if someone was born in 2025? The age would be negative!

üîç Complete Solution with Explanation:

def calculate_age(birth_year, current_year): """ Calculate a person's age based on their birth year Parameters: birth_year (int), current_year (int) Returns: age (int) """ # Simple subtraction gives us the age return current_year - birth_year # Test the function with different scenarios age1 = calculate_age(2005, 2024) # Someone born in 2005 age2 = calculate_age(2008, 2024) # Someone born in 2008 age3 = calculate_age(2000, 2024) # Someone born in 2000 # Display results with clear messages print(f"If you were born in 2005, you are {age1} years old.") print(f"If you were born in 2008, you are {age2} years old.") print(f"If you were born in 2000, you are {age3} years old.") # Bonus: Calculate your own age! my_age = calculate_age(2010, 2024) # Change 2010 to your birth year print(f"My age: {my_age} years old")
üéì What You Learned:
  • How to perform mathematical operations in functions
  • How parameters work with real-world data

  • Step 1: Calculate: area = length * width
  • Step 2: Format: f"A rectangle with length {length}..."
  • Step 3: Include all three values: length, width, and calculated area
Pro Tip: You can do the calculation directly in the f-string: f"...area of {length * width}..."
