üö® 5. Troubleshooting and Error Handling

5.1 Error Types & Detection

Syntax Error

Code violates Python grammar rules

if x > 10 # Missing colon

Logic Error

Program runs but gives wrong output

area = 2 * (length + width) # Wrong formula

Runtime Error

Error occurs during execution

result = 10 / 0 # ZeroDivisionError

5.2 Exception Handling

Exception Structure:

try:
# Risky code
except ExceptionType:
# Handle specific error
else:
# No exception occurred
finally:
# Always executes

Common Exceptions:

ValueError - Invalid value for operation
TypeError - Wrong data type
ZeroDivisionError - Division by zero
FileNotFoundError - File doesn't exist
IndexError - List index out of range

5.3 Unit Testing

unittest Framework:

class TestMath(unittest.TestCase):
def test_addition(self):
self.assertEqual(2+2, 4)

Assert Methods:

assertEqual(a, b) - a == b
assertTrue(x) - x is True
assertFalse(x) - x is False
assertIn(item, list) - item in list
assertIsInstance(obj, class) - type check
