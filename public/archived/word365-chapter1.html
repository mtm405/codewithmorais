Lab 1: Navigate Within Documents

Beginner

üé¨ Real-World Scenario

You're working on a 50-page employee handbook and need to quickly navigate between sections, find specific content, and move efficiently through the document without scrolling endlessly.

üìö Step-by-Step Instructions

1
Open Navigation Pane
View ‚Üí Navigation Pane

Click the Navigation Pane checkbox to display the document outline. This shows headings and allows quick jumping between sections.

2
Use Find & Replace
Home ‚Üí Find (Ctrl+F)

Use the Find function to locate specific text. Try advanced options like "Find All" to see every instance highlighted.

3
Go To Specific Pages
Home ‚Üí Go To (Ctrl+G)

Use Go To dialog to jump to specific pages, sections, lines, or bookmarks instantly.

4
Navigation Shortcuts
Keyboard Shortcuts

Practice: Ctrl+Home (document start), Ctrl+End (document end), Ctrl+Page Up/Down (page navigation).

üí° Pro Tips

Use Ctrl+F to search, then use F3 to find the next instance and Shift+F3 to find the previous instance. The Navigation Pane also shows search results with context.

‚úÖ Checkpoint

Practice navigating through a sample document using all methods. Can you find specific text and jump to different sections quickly?

Lab 2: Adding Bookmarks

Beginner

üé¨ Real-World Scenario

You're creating a training manual with multiple sections that readers will reference frequently. Bookmarks will allow quick access to important sections like "Safety Procedures," "Emergency Contacts," and "Training Schedules."

üìö Step-by-Step Instructions

1
Insert Bookmarks
Insert ‚Üí Bookmark

Position cursor at the location you want to bookmark, click Insert ‚Üí Bookmark, give it a descriptive name (no spaces), and click Add.

2
Navigate Using Bookmarks
Insert ‚Üí Bookmark ‚Üí Go To

Use the Bookmark dialog to select and go to any bookmark instantly. Alternatively, use Ctrl+G and select Bookmark from the list.

3
Create Bookmark Links
Insert ‚Üí Link ‚Üí Place in This Document

Create hyperlinks that jump to bookmarks. Select text, insert link, choose "Place in This Document," and select your bookmark.

4
Manage Bookmarks
Insert ‚Üí Bookmark

View all bookmarks, sort them, delete unnecessary ones, and ensure proper naming conventions for easy identification.

üí° Pro Tips

Use descriptive bookmark names like "Chapter_3_Safety" instead of "Bookmark1". Bookmarks are invisible unless you show them via File ‚Üí Options ‚Üí Advanced ‚Üí Show bookmarks.

‚úÖ Checkpoint

Create at least 3 bookmarks in your practice document and create a table of contents with hyperlinks to each bookmark.

Lab 6: Insert and Modify Headers and Footers

Intermediate

üé¨ Real-World Scenario

You're creating a business report that needs company logo in the header, page numbers in the footer, different first page header, and confidentiality notice on every page except the first.

üìö Step-by-Step Instructions

1

2
Add Footer Elements

3

üí° Pro Tips

‚úÖ Checkpoint

Lab 8: Saving Documents (Macro-enabled)

Advanced

üé¨ Real-World Scenario

Your department uses automated templates with macros for generating reports. You need to understand different save formats, macro security, and when to use macro-enabled documents.

üìö Step-by-Step Instructions

1
Understanding File Formats
File ‚Üí Save As ‚Üí Browse

Learn differences: .docx (standard), .docm (macro-enabled), .dotx (template), .dotm (macro template), .pdf, .rtf

2
Save as Macro-Enabled
File ‚Üí Save As ‚Üí Word Macro-Enabled Document (.docm)

When your document contains macros or VBA code, save in .docm format to preserve functionality.

3
Macro Security Settings
File ‚Üí Options ‚Üí Trust Center ‚Üí Macro Settings

Understand macro security levels: Disable all, Disable with notification, Enable all (not recommended).

4
Save Options & AutoRecover
File ‚Üí Options ‚Üí Save

Configure AutoRecover frequency, default save location, and file format preferences for efficiency.

‚ö†Ô∏è Security Warning

Only enable macros from trusted sources. Malicious macros can harm your computer. Always scan macro-enabled files with antivirus software.

‚úÖ Checkpoint

Practice saving documents in different formats. Understand when to use each format and configure appropriate macro security settings.

Lab 9: Document Properties & Accessibility

Intermediate

üé¨ Real-World Scenario

Your organization requires all documents to meet accessibility standards for users with disabilities and have proper metadata for document management systems.

üìö Step-by-Step Instructions

1
Document Properties
File ‚Üí Info ‚Üí Properties

Add title, author, subject, keywords, and comments. This metadata helps with document organization and searching.

2
Accessibility Checker
Review ‚Üí Check Accessibility

Run accessibility checker to identify issues like missing alt text, poor color contrast, or improper heading structure.

3
Add Alt Text to Images
Right-click image ‚Üí Alt Text

Provide meaningful descriptions for images, charts, and graphics so screen readers can describe them to visually impaired users.

4
Compatibility Check
File ‚Üí Info ‚Üí Check for Issues ‚Üí Check Compatibility

Ensure document works with older Word versions. Identify features that may not be supported in earlier versions.

üí° Accessibility Best Practices

Use heading styles for structure, sufficient color contrast (4.5:1 ratio), descriptive link text, and logical reading order. This helps all users, not just those with disabilities.

‚úÖ Checkpoint

Complete document properties, run accessibility checker, fix all identified issues, and verify compatibility with Word 2016 or later.
