Lab 3: Convert Tables to Text

Beginner

üé¨ Real-World Scenario

You need to export table data for use in another application that doesn't support tables, or you want to create a comma-separated values (CSV) file for database import.

üìö Step-by-Step Instructions

1
Select Table
Click table selector or select entire table

Click the table move handle (four-way arrow) in the top-left corner of the table to select the entire table.

2
Convert to Text
Table Layout ‚Üí Convert to Text

Choose how to separate the data: Paragraph marks (line breaks), Tabs, Commas, or Other separator.

3
Choose Separator Wisely
Separator Options

Tabs: Good for spreadsheets. Commas: Creates CSV format. Paragraphs: Each cell on new line.

4
Clean Up Text
Review and edit converted text

Review the converted text for formatting issues, extra spaces, or alignment problems that may need manual cleanup.

üí° Best Practices for Table Conversion

Commas: Perfect for CSV files and database imports. Tabs: Ideal for copying to Excel. Paragraphs: Good for simple lists. Choose separator based on your intended use.

‚úÖ Checkpoint

Convert a sample table to comma-separated text, then to tab-separated text. Compare the results and understand when to use each format.

Intermediate

üé¨ Real-World Scenario

You have a table with employee information including names, departments, hire dates, and salaries. You need to sort this data by different criteria: alphabetically by name, by department, or by hire date to create different reports.

üìö Step-by-Step Instructions

1

Lab 5: Modify Table Layout

Advanced

üé¨ Real-World Scenario

Your initial table design needs adjustments: add columns for new data fields, merge cells for category headers, adjust column widths for better readability, and modify row heights to accommodate longer text.

üìö Step-by-Step Instructions

1
Insert/Delete Rows and Columns
Table Layout ‚Üí Insert Above/Below, Insert Left/Right

Right-click for quick insert/delete options, or use ribbon commands. Insert rows above/below cursor, columns left/right of cursor.

2
Merge and Split Cells
Select cells ‚Üí Table Layout ‚Üí Merge Cells / Split Cells

Merge cells for headers spanning multiple columns, or split cells to create subcategories within a column.

3
Adjust Column Width and Row Height
Drag borders or Table Layout ‚Üí Cell Size

Drag column borders for visual adjustment, or use ribbon for exact measurements. AutoFit options resize to content or window.

4
Table Properties
Right-click ‚Üí Table Properties

Control table alignment, text wrapping around table, borders, and advanced cell spacing options.

üí° Layout Best Practices

AutoFit to Contents: Adjusts column width to fit text. AutoFit to Window: Stretches table across page width. Fixed Width: Maintains specific measurements regardless of content.

‚úÖ Checkpoint

Modify a table by adding rows/columns, merging cells for a header, adjusting column widths proportionally, and setting a specific row height for better appearance.
