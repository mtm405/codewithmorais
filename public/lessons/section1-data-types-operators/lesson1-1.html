Theory: Understanding Python Data Types

Python automatically assigns data types to variables based on the values they store. Understanding these data types is fundamental to writing effective Python programs.

üîç What You'll Learn:
  • How Python determines data types automatically
  • The four fundamental data types: str, int, float, bool
  • How to check data types using the type() function
  • When and why each data type is used

1. String (str) - Text Data

Strings represent text data and are created by enclosing text in quotes.

String Examples
# Different ways to create strings
name = "Alice"              # Double quotes
message = 'Hello World'     # Single quotes
paragraph = """This is a
multi-line string"""        # Triple quotes

# Check the type
print(type(name))           # Output: <class 'str'>
print(type(message))        # Output: <class 'str'>

2. Integer (int) - Whole Numbers

Integers represent whole numbers without decimal points.

Integer Examples
# Integer assignments
age = 25                    # Positive integer
temperature = -5            # Negative integer
count = 0                   # Zero

# Check the type
print(type(age))            # Output: <class 'int'>

# Large integers are supported
big_number = 123456789012345
print(type(big_number))     # Output: <class 'int'>

3. Float (float) - Decimal Numbers

Floats represent numbers with decimal points.

Float Examples
# Float assignments
price = 19.99               # Standard decimal
pi = 3.14159                # Mathematical constant
zero_float = 0.0            # Zero as float

# Scientific notation
scientific = 1.5e-4         # 0.00015

# Check the type
print(type(price))          # Output: <class 'float'>
print(type(scientific))     # Output: <class 'float'>

4. Boolean (bool) - True/False Values

Booleans represent truth values and are either True or False.

Boolean Examples
# Boolean assignments
is_student = True           # Capitalized True
is_graduated = False        # Capitalized False

# Booleans from comparisons
is_adult = age >= 18        # Results in True or False
is_equal = 5 == 5          # Results in True

# Check the type
print(type(is_student))     # Output: <class 'bool'>
print(type(is_equal))       # Output: <class 'bool'>
