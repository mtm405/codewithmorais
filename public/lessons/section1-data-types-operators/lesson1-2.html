Type Conversion

Type conversion (casting) allows you to change one data type to another. Python provides built-in functions for common conversions:

  • int() - Converts to integer
  • float() - Converts to floating-point number
  • str() - Converts to string
  • bool() - Converts to boolean
# Type conversion examples # String to integer age_str = "25" age_int = int(age_str) print(f"String: {age_str}, Integer: {age_int}") # Integer to string score = 95 score_str = str(score) print(f"Integer: {score}, String: '{score_str}'") # String to float price_str = "19.99" price_float = float(price_str) print(f"String: {price_str}, Float: {price_float}") # Various to boolean print(f"bool(1): {bool(1)}") print(f"bool(0): {bool(0)}") print(f"bool('hello'): {bool('hello')}") print(f"bool(''): {bool('')}")

String Indexing & Slicing

Strings in Python are sequences, which means you can access individual characters using indexing and extract portions using slicing:

  • Indexing: Access single characters using square brackets
  • Negative indexing: Count from the end using negative numbers
  • Slicing: Extract substrings using start:end notation
# String indexing and slicing text = "Python Programming" # Indexing (0-based) print(f"First character: {text[0]}") print(f"Character at index 7: {text[7]}") # Negative indexing print(f"Last character: {text[-1]}") print(f"Second to last: {text[-2]}") # Slicing [start:end] print(f"First 6 characters: {text[0:6]}") print(f"Characters 7-17: {text[7:18]}") print(f"From index 7 to end: {text[7:]}") print(f"First 6 characters: {text[:6]}") # Step slicing [start:end:step] print(f"Every 2nd character: {text[::2]}") print(f"Reverse string: {text[::-1]}")

Introduction to Lists

Lists are ordered collections that can store multiple items. They are mutable (can be changed) and allow duplicate values:

  • Creation: Use square brackets [] or list() function
  • Indexing: Access elements like strings
  • Methods: append(), remove(), insert(), etc.
# List creation and operations # Creating lists fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5] mixed = ["hello", 42, True, 3.14] # Accessing elements print(f"First fruit: {fruits[0]}") print(f"Last number: {numbers[-1]}") # List methods fruits.append("grape") # Add to end print(f"After append: {fruits}") fruits.insert(1, "kiwi") # Insert at position print(f"After insert: {fruits}") fruits.remove("banana") # Remove by value print(f"After remove: {fruits}") # List slicing print(f"First 3 fruits: {fruits[:3]}") print(f"Numbers from index 2: {numbers[2:]}") # List length print(f"Number of fruits: {len(fruits)}") print(f"Number of numbers: {len(numbers)}")
