Arithmetic Operators

Basic Mathematical Operations

Arithmetic operators perform mathematical calculations on numeric values.

+ Addition

Adds two numbers or concatenates strings

- Subtraction

Subtracts the second number from the first

* Multiplication

Multiplies two numbers or repeats strings

/ Division

Divides first number by second (float result)

// Floor Division

Divides and rounds down to nearest integer

% Modulo

Returns remainder after division

** Exponentiation

Raises first number to power of second

# Arithmetic operators examples a, b = 10, 3 print(f"{a} + {b} = {a + b}") # Addition: 13 print(f"{a} - {b} = {a - b}") # Subtraction: 7 print(f"{a} * {b} = {a * b}") # Multiplication: 30 print(f"{a} / {b} = {a / b}") # Division: 3.333... print(f"{a} // {b} = {a // b}") # Floor division: 3 print(f"{a} % {b} = {a % b}") # Modulo: 1 print(f"{a} ** {b} = {a ** b}") # Exponentiation: 1000 # String operations text = "Python" print(f"'{text}' * 3 = {'Python' * 3}") # Repetition print(f"'Hello' + ' World' = {'Hello' + ' World'}") # Concatenation

Comparison Operators

Comparing Values

Comparison operators compare two values and return a boolean result (True or False).

== Equal to

Checks if two values are equal

!= Not equal to

Checks if two values are different

< Less than

Checks if left value is smaller

> Greater than

Checks if left value is larger

<= Less than or equal

Checks if left is smaller or equal

>= Greater than or equal

Checks if left is larger or equal

# Comparison operators examples x, y = 5, 10 print(f"{x} == {y}: {x == y}") # False print(f"{x} != {y}: {x != y}") # True print(f"{x} < {y}: {x < y}") # True print(f"{x} > {y}: {x > y}") # False print(f"{x} <= {y}: {x <= y}") # True print(f"{x} >= {y}: {x >= y}") # False # String comparison (lexicographic) print(f"'apple' < 'banana': {'apple' < 'banana'}") # True print(f"'Python' == 'Python': {'Python' == 'Python'}") # True # Chained comparisons age = 25 print(f"18 <= {age} <= 65: {18 <= age <= 65}") # True

Logical Operators

Boolean Logic

Logical operators combine or modify boolean expressions.

and Logical AND

Returns True if both operands are True

or Logical OR

Returns True if at least one operand is True

not Logical NOT

Returns the opposite boolean value

# Logical operators examples a, b = True, False print(f"{a} and {b}: {a and b}") # False print(f"{a} or {b}: {a or b}") # True print(f"not {a}: {not a}") # False print(f"not {b}: {not b}") # True # Practical example age = 25 has_license = True has_insurance = False can_drive = age >= 18 and has_license print(f"Can drive: {can_drive}") # True is_insured = has_license and has_insurance print(f"Is insured: {is_insured}") # False # Short-circuit evaluation print(f"False and (anything): {False and (1/0 if False else True)}") # False (doesn't evaluate 1/0)

Assignment Operators

Variable Assignment

Assignment operators assign values to variables, often with an operation.

= Basic Assignment

Assigns value to variable

+= Add and Assign

Adds value and assigns result

-= Subtract and Assign

Subtracts value and assigns result

*= Multiply and Assign

Multiplies value and assigns result

/= Divide and Assign

Divides value and assigns result

%= Modulo and Assign

Applies modulo and assigns result

# Assignment operators examples x = 10 print(f"Initial x: {x}") x += 5 # Equivalent to x = x + 5 print(f"After x += 5: {x}") # 15 x -= 3 # Equivalent to x = x - 3 print(f"After x -= 3: {x}") # 12 x *= 2 # Equivalent to x = x * 2 print(f"After x *= 2: {x}") # 24 x /= 4 # Equivalent to x = x / 4 print(f"After x /= 4: {x}") # 6.0 x %= 4 # Equivalent to x = x % 4 print(f"After x %= 4: {x}") # 2.0 # String concatenation with += message = "Hello" message += " World" print(f"Message: {message}") # Hello World

Identity & Membership Operators

Identity Operators

Identity operators check if two variables refer to the same object in memory.

is Identity

Returns True if both variables are the same object

is not Not Identity

Returns True if variables are different objects

Membership Operators

Membership operators check if a value exists in a sequence.

in Membership

Returns True if value is found in sequence

not in Not Membership

Returns True if value is not in sequence

# Identity operators examples a = [1, 2, 3] b = [1, 2, 3] c = a print(f"a == b: {a == b}") # True (same content) print(f"a is b: {a is b}") # False (different objects) print(f"a is c: {a is c}") # True (same object) print(f"a is not b: {a is not b}") # True # Membership operators examples fruits = ["apple", "banana", "orange"] print(f"'apple' in fruits: {'apple' in fruits}") # True print(f"'grape' in fruits: {'grape' in fruits}") # False print(f"'grape' not in fruits: {'grape' not in fruits}") # True # String membership text = "Python Programming" print(f"'Python' in text: {'Python' in text}") # True print(f"'Java' not in text: {'Java' not in text}") # True # Number ranges numbers = range(1, 11) print(f"5 in range(1, 11): {5 in numbers}") # True print(f"15 in range(1, 11): {15 in numbers}") # False
