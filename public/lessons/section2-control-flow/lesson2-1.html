Understanding Conditional Statements

Conditional statements allow your program to make decisions and execute different code blocks based on specific conditions. They are the foundation of program logic and enable dynamic behavior.

Key Concepts:

  • Conditional statements evaluate boolean expressions
  • Code blocks are executed based on True/False results
  • Indentation is crucial in Python to define code blocks
  • Multiple conditions can be chained using elif

The if Statement

The if statement is the most basic conditional statement. It executes a block of code only if a specified condition is True.

Syntax:

if condition: # Code to execute if condition is True statement1 statement2 # ...

Important: Python uses indentation (typically 4 spaces) to define code blocks. All statements at the same indentation level belong to the same block.

# Basic if statement examples age = 18 if age >= 18: print("You are an adult!") print("You can vote.") # Multiple conditions temperature = 25 if temperature > 30: print("It's hot outside!") if temperature < 0: print("It's freezing!") # Using variables in conditions has_license = True is_insured = True if has_license: print("You can drive!") if has_license and is_insured: print("You're fully legal to drive!")

The if-else Statement

The else statement provides an alternative path when the if condition is False. It ensures that one of the two code blocks will always execute.

# if-else statement syntax if condition: # Code to execute if condition is True else: # Code to execute if condition is False
# if-else examples age = 16 if age >= 18: print("You are an adult!") print("You can vote and drive.") else: print("You are a minor.") print("Wait a few more years!") # Grade evaluation score = 85 if score >= 60: print("Congratulations! You passed!") print(f"Your score: {score}") else: print("Sorry, you need to retake the exam.") print(f"Your score: {score} (minimum: 60)") # Number classification number = -5 if number > 0: print(f"{number} is positive") else: print(f"{number} is negative or zero")

The if-elif-else Chain

When you need to check multiple conditions in sequence, use elif (else if). Python checks each condition in order and executes the first True block it finds.

# if-elif-else syntax if condition1: # Code for condition1 elif condition2: # Code for condition2 elif condition3: # Code for condition3 else: # Code if none of the above conditions are True
# Grade classification system score = 87 if score >= 90: grade = "A" print(f"Excellent! Grade: {grade}") elif score >= 80: grade = "B" print(f"Good job! Grade: {grade}") elif score >= 70: grade = "C" print(f"Average. Grade: {grade}") elif score >= 60: grade = "D" print(f"Below average. Grade: {grade}") else: grade = "F" print(f"Failed. Grade: {grade}") # Weather conditions temperature = 22 if temperature > 30: print("It's hot! Wear light clothes.") elif temperature > 20: print("It's warm. Perfect weather!") elif temperature > 10: print("It's cool. Bring a light jacket.") elif temperature > 0: print("It's cold. Wear warm clothes.") else: print("It's freezing! Bundle up!") # Traffic light system light_color = "yellow" if light_color == "green": print("Go!") elif light_color == "yellow": print("Caution - prepare to stop") elif light_color == "red": print("Stop!") else: print("Invalid light color")

Nested Conditionals

You can place conditional statements inside other conditional statements to create more complex decision trees. This is called nesting.

# Nested conditional examples age = 25 has_license = True has_car = False if age >= 18: print("You are an adult.") if has_license: print("You have a driving license.") if has_car: print("You can drive your own car!") else: print("You can rent a car or use car sharing.") else: print("You need to get a driving license.") else: print("You are too young to drive.") # Bank account access account_balance = 1000 pin_correct = True card_inserted = True if card_inserted: print("Card detected.") if pin_correct: print("PIN verified.") if account_balance > 0: print(f"Welcome! Your balance: ${account_balance}") else: print("Insufficient funds.") else: print("Incorrect PIN. Access denied.") else: print("Please insert your card.")
