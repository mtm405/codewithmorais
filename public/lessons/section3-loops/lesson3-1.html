Theory

What are For Loops?

A for loop is used to repeat a block of code for each item in a sequence (like a list, string, or range).

Basic Syntax:

for item in sequence:
    # code to repeat

Key Concepts:

  • Iteration: Going through each item one by one
  • Range: Creating sequences of numbers
  • Lists: Looping through list elements
  • Strings: Looping through characters

Loop through a list:

fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

Loop with range:

for i in range(5):
    print(f"Number: {i}")
