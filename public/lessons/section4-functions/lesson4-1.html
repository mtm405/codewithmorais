Theory

What are Functions?

A function is a reusable block of code that performs a specific task. Think of it as a mini-program inside your program!

Basic Function Syntax:

def function_name():
    # code to execute
    return result # optional

Key Concepts:

  • def: Keyword to define a function
  • Function name: What you call to use the function
  • Parentheses: Where parameters go (more on this later)
  • Indentation: Code inside the function
  • return: Send a value back (optional)

Simple Function:

def greet():
    print("Hello, World!")

# Call the function
greet()

Function with Return:

def get_greeting():
    return "Hello, Python!"

# Use the returned value
message = get_greeting()
print(message)

Why Use Functions?

  • Reusability: Write once, use many times
  • Organization: Break large problems into smaller pieces
  • Maintainability: Easier to fix and update
  • Testing: Test individual pieces separately
