üìÑDomain 1: Manage Documents

This domain focuses on document settings, navigation, and properties.

Navigate within documents

  • Search for text: Use Find to locate specific content quickly
    Home Tab ‚Üí Editing Group ‚Üí Find Ctrl+F
  • Go To specific locations: Jump to pages, sections, bookmarks, or other objects
    Home Tab ‚Üí Editing Group ‚Üí Go To Ctrl+G
  • Insert Bookmarks: Create named locations for easy navigation
    Insert Tab ‚Üí Links Group ‚Üí Bookmark
  • Insert Hyperlinks: Link to websites, files, or locations within the document
    Insert Tab ‚Üí Links Group ‚Üí Hyperlink Ctrl+K
üí° Pro Tip: Use bookmarks for long documents to quickly jump between sections. Combine with hyperlinks for an interactive table of contents!

Format documents

  • Page Setup: Configure orientation (Portrait/Landscape), margins, and page size
    Layout Tab ‚Üí Page Setup Group
  • Headers and Footers: Insert and modify content that appears on every page
    Insert Tab ‚Üí Header & Footer Group
  • Page Background: Add watermarks, page color, or page borders
    Design Tab ‚Üí Page Background Group
  • Style Sets: Apply coordinated formatting themes to your document
    Design Tab ‚Üí Document Formatting Group
üí° Pro Tip: Different headers/footers for odd and even pages can make your document look more professional!

Save and share documents

  • Save in alternative formats: Export to PDF, XPS, or other formats
    File Tab ‚Üí Save As
  • Document Properties: Edit title, author, subject, and other metadata
    File Tab ‚Üí Info ‚Üí Properties
  • Print Settings: Configure pages, copies, orientation, and printer options
    File Tab ‚Üí Print Ctrl+P
  • Share Options: Share via email, OneDrive, or other methods
    File Tab ‚Üí Share

Inspect documents for issues

  • Document Inspector: Find and remove hidden properties, comments, and personal information
    File Tab ‚Üí Info ‚Üí Check for Issues ‚Üí Inspect Document
  • Accessibility Checker: Identify accessibility issues for users with disabilities
    File Tab ‚Üí Info ‚Üí Check for Issues ‚Üí Check Accessibility
  • Compatibility Checker: Find features not supported in earlier Word versions
    File Tab ‚Üí Info ‚Üí Check for Issues ‚Üí Check Compatibility
üí° Pro Tip: Always run Document Inspector before sharing sensitive documents to remove hidden metadata!

üë•Domain 6: Manage Document Collaboration

This domain focuses on reviewing and tracking changes in collaborative documents.

Add and manage comments

  • Add Comments: Insert comments to provide feedback without changing the document

  • Review Comments: Navigate through all comments in the document

    Review Tab ‚Üí Changes Group ‚Üí Accept
  • Reject Changes: Discard tracked changes and keep original content
    Review Tab ‚Üí Changes Group ‚Üí Reject
  • Lock Tracking: Prevent others from turning off Track Changes
    Review Tab ‚Üí Tracking Group ‚Üí Lock Tracking
  • Show Markup: Choose which types of markup to display (comments, insertions, deletions, formatting)
    Review Tab ‚Üí Tracking Group ‚Üí Show Markup
üí° Pro Tip: Lock tracking with a password to ensure all reviewers' changes are recorded!

üìù MOS Word Associate Practice Questions

1. What keyboard shortcut opens the Find feature to search for text in a document?

  • A) Ctrl+H
  • B) Ctrl+F
  • C) Ctrl+G
  • D) Ctrl+K
Explanation: Ctrl+F opens the Find feature. Ctrl+H is Replace, Ctrl+G is Go To, and Ctrl+K is Insert Hyperlink.

2. Which tab contains the Document Inspector tool?

  • A) Review Tab
  • B) Home Tab
  • C) File Tab (Backstage View)
  • D) Insert Tab
Explanation: Document Inspector is located in File Tab ‚Üí Info ‚Üí Check for Issues ‚Üí Inspect Document. It removes hidden properties and personal information.

3. To copy formatting from one location and apply it elsewhere, you should use:

  • A) Format Painter
  • B) Text Effects
  • C) Styles
  • D) Clear Formatting
Explanation: Format Painter (Home Tab ‚Üí Clipboard Group) copies formatting. Double-click it to apply formatting to multiple locations!

4. Which keyboard shortcut inserts a page break?

  • A) Ctrl+B
  • B) Ctrl+Shift+Enter
  • C) Alt+Enter
  • D) Ctrl+Enter
Explanation: Ctrl+Enter inserts a page break, forcing content to start on a new page. You can also access this via Layout Tab ‚Üí Page Setup ‚Üí Breaks.

5. To automatically display a table's header row on each page, you should use:

  • A) Merge Cells
  • B) Repeat Header Rows
  • C) Split Table
  • D) Sort Table
Explanation: Repeat Header Rows (Table Tools Layout Tab ‚Üí Table Group) automatically shows the header row at the top of each page when a table spans multiple pages.

6. Which tab contains the Table of Contents feature?

  • A) References Tab
  • B) Insert Tab
  • C) Design Tab
  • D) Layout Tab
Explanation: Table of Contents is in References Tab ‚Üí Table of Contents Group. It automatically generates based on heading styles in your document.

7. To remove the background from a picture, which tool should you use?

  • A) Artistic Effects
  • B) Picture Effects
  • C) Remove Background
  • D) Crop
Explanation: Remove Background (Picture Tools Format Tab ‚Üí Adjust Group) automatically detects and removes the background from images.

8. What keyboard shortcut enables Track Changes?

  • A) Ctrl+Alt+M
  • B) Ctrl+T
  • C) Alt+Shift+E
  • D) Ctrl+Shift+E
Explanation: Ctrl+Shift+E toggles Track Changes on/off. Track Changes records all edits made to the document for review.

9. To insert a footnote, you should press:

  • A) Alt+Ctrl+D
  • B) Alt+Ctrl+F
  • C) Ctrl+Shift+F
  • D) Alt+F
Explanation: Alt+Ctrl+F inserts a footnote (appears at page bottom), while Alt+Ctrl+D inserts an endnote (appears at document end).

10. Which feature allows you to control how text flows around images?

  • A) Position
  • B) Alt Text
  • C) Wrap Text
  • D) Align
Explanation: Wrap Text (Format Tab ‚Üí Arrange Group) controls text flow with options like Square, Tight, Through, Top and Bottom, etc.

11. To convert comma-separated text into a table, which feature should you use?

  • A) Convert Text to Table
  • B) Insert Table
  • C) Quick Tables
  • D) Draw Table
Explanation: Convert Text to Table (Insert Tab ‚Üí Tables Group) transforms delimited text into a table structure based on commas, tabs, or other separators.

12. Which view option shows all tracked changes with detailed markup?

  • A) Simple Markup
  • B) No Markup
  • C) Original
  • D) All Markup
Explanation: All Markup (Review Tab ‚Üí Tracking Group ‚Üí Display for Review) shows every change with full details including insertions, deletions, and formatting.
