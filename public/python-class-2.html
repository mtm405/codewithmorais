üìù Detailed Information and Examples

A. Branching Statements (Objective 2.1)

Python uses if, elif, and else for decision making, supporting nested and compound conditional expressions.

Example 1: Basic and Compound Branching
score = 85 # Basic Branching if score >= 90: print("A") elif score >= 80 and score < 90: # Compound conditional expression print("B") else: print("C") # Output: B # Nested Branching is_premium = True if score > 70: if is_premium: print("Congrats, Premium!") # Nested check else: print("Congrats!") # Output: Congrats, Premium!

B. Iteration Statements (Objective 2.2)

Python uses while (for conditional repetition) and for (for sequence iteration).

Example 2: for loop and while loop
# for loop (Iterating over a sequence/list) my_list = ['apple', 'banana', 'cherry'] for item in my_list: print(f"Fruit: {item}") # Output: # Fruit: apple # Fruit: banana # Fruit: cherry # while loop (Conditional repetition) count = 0 while count < 3: print(f"Count is: {count}") count += 1 # Output: Count is: 0, 1, 2
Example 3: break, continue, and pass
numbers = [1, 2, 3, 4, 5, 6] for num in numbers: if num % 2 != 0: continue # Skip odd numbers (1, 3, 5) if num > 5: break # Stop loop when we hit 6 print(f"Even number: {num}") # Output: # Even number: 2 # Even number: 4 # The pass statement does nothing def placeholder_function(): pass # Use pass as a placeholder for future code

‚ùì Quiz: Review Check

Question 1: What is the purpose of the elif statement in Python?

  • a) To end an if statement
  • b) To execute code when the initial if condition is False
  • c) To check a second condition only if the previous if or elif conditions were False ‚úì
  • d) To force an error
Answer: c)
elif allows checking multiple conditions in sequence. It only runs if all previous conditions were False.

Question 2: Which statement is used to immediately exit a loop?

  • a) continue
  • b) pass
  • c) exit()
  • d) break ‚úì
Answer: d) break
break immediately exits the innermost loop. continue skips to the next iteration.

Question 3: What is the output of the following nested loop?

for i in range(2): for j in range(1): print(i * j)
  • a) 0, 0 ‚úì
  • b) 0, 1
  • c) 0, 1, 2
  • d) 0, 1, 0, 1
Answer: a) 0, 0
Outer loop runs twice (i=0, i=1). Inner loop runs once each time (j=0).
First iteration: 0 * 0 = 0
Second iteration: 1 * 0 = 0

Question 4: If a condition in a while loop is never met, what will occur?

  • a) A syntax error
  • b) An infinite loop ‚úì
  • c) The code will skip the loop entirely
  • d) A runtime error
Answer: b) An infinite loop
If the loop condition never becomes False, the loop will run forever (infinite loop).

Question 5: The keyword pass can be used as a temporary placeholder in which flow control structures?

  • a) if/else statements
  • b) while loops
  • c) for loops
  • d) All of the above ‚úì
Answer: d) All of the above
pass is a null operation that can be used anywhere a statement is syntactically required but no action is needed.
