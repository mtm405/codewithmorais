üéØ Part 1: Interacting with the System (Modules 6.1)

This section focuses on the os, sys, os.path, and io modules‚Äîyour tools for interacting with the operating system and handling files.

üìù Core System Modules

Python provides several built-in modules that allow your programs to interact with the operating system, read command-line arguments, and manage files efficiently. Think of these modules as your program's "bridge" to the computer's file system and environment.

Why These Modules Matter:

  • ‚Ä¢ Automation: Create scripts that organize files, backup data, or clean up directories automatically
  • ‚Ä¢ Portability: Write code that works on Windows, Mac, and Linux without changes
  • ‚Ä¢ Professional Tools: Build command-line applications that users can run with different settings
  • ‚Ä¢ File Safety: Check if files exist before trying to read them (prevents crashes)
Module Role Key Uses
os Operating System Interface Creating folders (os.mkdir()), changing directories (os.chdir()), running shell commands
os.path Path Management Checking if files/folders exist (os.path.exists()), joining paths (os.path.join())
sys System Parameters & Arguments Accessing command-line arguments (sys.argv), exiting the program (sys.exit())
io Input/Output Stream Handling Underlying module used when you open() and read/write files
Example: Basic OS Operations

Scenario: You're building a file organizer that needs to know where it is and create folders for different file types.

import os # Get current working directory (where your script is running) current_dir = os.getcwd() print(f"Current directory: {current_dir}") # Example output: C:\Users\Student\Documents # Create a new directory for organizing files os.mkdir("my_new_folder") print("‚úÖ Created 'my_new_folder'") # Change to the new directory os.chdir("my_new_folder") print(f"Now in: {os.getcwd()}") # List all files in current directory files = os.listdir() print(f"Files in this folder: {files}") # Empty folder will show: []

üí° Common Pitfall: If you run os.mkdir("my_new_folder") twice, you'll get a FileExistsError. Always check if the folder exists first using os.path.exists()!

üîç Understanding File Paths

Absolute Path: The complete path from the root directory.

C:\Users\Student\Documents\project\data.txt (Windows)

/home/student/documents/project/data.txt (Mac/Linux)

Relative Path: Path relative to your current location.

./data.txt (current folder)

../data.txt (parent folder)

üöÄ Command-Line Arguments (sys.argv)

The sys module lets your Python script take information directly from the command line when it's executed. This is how professional tools work‚Äîthey accept different inputs without modifying the code!

Real-World Examples:

  • ‚Ä¢ git commit -m "My message" (git uses arguments)
  • ‚Ä¢ python backup.py folder1 folder2 (backup specific folders)
  • ‚Ä¢ python grader.py student_name 95 (grade calculator)

How sys.argv Works:

  • ‚Ä¢ sys.argv is a list of strings containing everything typed after "python"
  • ‚Ä¢ sys.argv[0] is always the script name itself
  • ‚Ä¢ sys.argv[1] is the first argument, sys.argv[2] is the second, and so on
  • ‚Ä¢ Use len(sys.argv) to check how many arguments were provided
Visual Breakdown: sys.argv

Command: python calculator.py add 10 5

sys.argv contains:

sys.argv[0] = "calculator.py" (script name)

sys.argv[1] = "add" (operation)

sys.argv[2] = "10" (first number)

sys.argv[3] = "5" (second number)

len(sys.argv) = 4 (total items)

‚ö†Ô∏è Important: Arguments are always strings! If you need numbers, convert them: int(sys.argv[2])

üî¢ The Math Module

The math module provides functions that go beyond standard arithmetic operators. It's essential for scientific calculations, rounding operations, and mathematical constants.

When You Need the Math Module:

  • ‚Ä¢ E-commerce: Rounding prices correctly (ceil() for "round up to nearest dollar")
  • ‚Ä¢ Game Development: Calculating distances, angles, and trajectories
  • ‚Ä¢ Data Science: Statistical calculations and normalizations
  • ‚Ä¢ Engineering: Precise mathematical operations with constants like œÄ
üéØ Rounding Concepts Explained

Ceiling (math.ceil): "Round UP to the next whole number"

Think: Buying ceiling tiles. If you need 4.2 tiles, you must buy 5 (can't buy 0.2 of a tile!)

Floor (math.floor): "Round DOWN to the previous whole number"

Think: Building floors. A 4.9-story building still only has 4 complete floors.

Truncate (math.trunc): "Chop off the decimal part"

Think: Cutting a rope. Just cut at the decimal point, keeping only the whole number part.

Function Purpose Example (x = 4.7)
math.ceil(x) Smallest integer ‚â• x (ceiling) math.ceil(4.7) is 5
math.floor(x) Largest integer ‚â§ x (floor) math.floor(4.7) is 4
math.trunc(x) Truncate: remove fractional part math.trunc(4.7) is 4
math.sqrt(x) Square root math.sqrt(16) is 4.0
math.isqrt(x) Integer square root math.isqrt(16) is 4
math.pi The value of pi (œÄ) 3.14159...
math.isnan(x) Check if x is Not a Number math.isnan(float('nan')) is True
‚ö†Ô∏è Important Distinction: trunc() vs floor()

For positive numbers: math.trunc(4.7) and math.floor(4.7) both return 4

For negative numbers: They differ!

  • ‚Ä¢ math.trunc(-4.7) returns -4 (cuts off decimal)
  • ‚Ä¢ math.floor(-4.7) returns -5 (rounds down to lowest integer)

‚è±Ô∏è The Datetime Module

The datetime module helps manage dates and times accurately. It's crucial for logging, scheduling, and time-based calculations.

Real-World Uses:

  • ‚Ä¢ Event Logging: Record exactly when errors occur in your application
  • ‚Ä¢ Scheduling: Determine if it's time to run a backup or send reminders
  • ‚Ä¢ Age Calculation: Calculate someone's age from their birthdate
  • ‚Ä¢ Time Zones: Handle international time differences in global apps
  • ‚Ä¢ Billing Systems: Track subscription start/end dates, calculate time used
üìÖ Understanding Datetime Components

A datetime object contains:

  • ‚Ä¢ Year: Full year (2025)
  • ‚Ä¢ Month: 1-12 (January = 1, December = 12)
  • ‚Ä¢ Day: 1-31 (depending on month)
  • ‚Ä¢ Hour: 0-23 (24-hour format)
  • ‚Ä¢ Minute: 0-59
  • ‚Ä¢ Second: 0-59
  • ‚Ä¢ Microsecond: 0-999999 (for precision timing)

Key Methods:

  • ‚Ä¢ Current Time: datetime.datetime.now() returns a datetime object with current date/time
  • ‚Ä¢ Day of Week: .weekday() returns the day as an integer (0=Monday, 6=Sunday)
  • ‚Ä¢ Formatting: .strftime(format_code) converts datetime to a formatted string
  • ‚Ä¢ Parsing: .strptime(string, format) converts a string back to datetime
Format Code Meaning Example
%Y Year with century 2025
%m Month as a number (01-12) 11
%d Day of the month (01-31) 03
%H Hour (24-hour clock) 14
%M Minute (00-59) 30
%S Second (00-59) 45
Example: Working with Dates and Times

Scenario: Event Logger - Track when users log in and format timestamps for reports.

import datetime # Get current date and time now = datetime.datetime.now() print(f"Current datetime: {now}") # Output: 2025-11-03 14:30:45.123456 # Access individual components print(f"Year: {now.year}") # 2025 print(f"Month: {now.month}") # 11 print(f"Day: {now.day}") # 3 print(f"Hour: {now.hour}") # 14 (2 PM in 24-hour format) # Get day of week (0=Monday, 6=Sunday) day_of_week = now.weekday() days = ['Monday', 'Tuesday', 'Wednesday', 'Thursday', 'Friday', 'Saturday', 'Sunday'] print(f"Today is {days[day_of_week]} (index: {day_of_week})") # Format for database (ISO format) db_format = now.strftime("%Y-%m-%d %H:%M:%S") print(f"Database format: {db_format}") # Output: 2025-11-03 14:30:45 # Format for users (readable format) user_format = now.strftime("%B %d, %Y at %I:%M %p") print(f"User-friendly: {user_format}") # Output: November 03, 2025 at 02:30 PM # Format for file names (no spaces or special chars) filename_format = now.strftime("backup_%Y%m%d_%H%M%S.zip") print(f"Filename: {filename_format}") # Output: backup_20251103_143045.zip

Scenario: Check if it's the weekend

üÉè The Random Module

The random module is crucial for simulations, games, and security. It provides various functions for generating random numbers and selecting random elements.

Real-World Applications:

  • ‚Ä¢ Gaming: Dice rolls, card shuffling, random enemy spawns, loot drops
  • ‚Ä¢ Simulations: Monte Carlo simulations, testing with random data
  • ‚Ä¢ Machine Learning: Random sampling of training data, initialization of neural networks
  • ‚Ä¢ Contests: Selecting random winners from a participant list
  • ‚Ä¢ A/B Testing: Randomly assigning users to different test groups
  • ‚Ä¢ Security: Generating random passwords, tokens, and session IDs
‚ö†Ô∏è Important: Pseudo-Random vs True Random

Python's random module generates pseudo-random numbers‚Äîthey appear random but follow a mathematical pattern. This is perfect for games and simulations but NOT for cryptography.

For security/cryptography: Use secrets module instead:

import secrets

secure_token = secrets.token_hex(16)

Function Purpose Example
random.randint(a, b) Integer, inclusive of both a and b randint(1, 10) ‚Üí 1, 2, ..., 10
random.choice(seq) Selects a single, random element from a list choice(['A', 'B', 'C']) ‚Üí 'B'
random.sample(pop, k) Selects k unique elements from a list sample(range(1, 50), 6) ‚Üí 6 unique numbers
random.shuffle(x) Shuffles the list in place (modifies the original) shuffle(my_list)
üí° Key Difference: choice() vs sample()

choice(): Returns ONE random element. Can select the same element multiple times if called repeatedly.

sample(): Returns MULTIPLE unique elements at once. No duplicates in the result.

Example: Random Module in Action

Question 1: How do you check if a file named config.ini exists?

Question 2: Which sys property holds the command-line arguments?

Answer: sys.argv
Explanation: sys.argv is a list containing command-line arguments. Index 0 is the script name, index 1+ are arguments.

Question 3: What does math.trunc(-5.2) return?

Module: math

Answer: -5
Explanation: math.trunc() removes the fractional part, so -5.2 becomes -5. Note: math.floor(-5.2) would return -6.

Answer: 4
Explanation: The weekday() method returns: 0=Monday, 1=Tuesday, 2=Wednesday, 3=Thursday, 4=Friday, 5=Saturday, 6=Sunday

Question 5: How do you select 5 unique winning numbers from a list of 50?

Module: random

Answer: random.sample(list, 5)
Explanation: random.sample(population, k) returns k unique elements from the population. Perfect for lottery-style selections.

Explanation: The format code %m represents the month as a zero-padded decimal number (01-12).

Question 7: What's the difference between math.ceil(4.2) and math.floor(4.2)?

Module: math

Explanation: ceil() rounds UP to the smallest integer ‚â• x. floor() rounds DOWN to the largest integer ‚â§ x.
