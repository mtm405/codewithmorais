1. Data Types & Type Conversion

üìå Understanding Python Data Types

Key Points:
  • Python does NOT require you to declare data types (dynamically typed)
  • Python distinguishes between integers (int) and floating-point numbers (float)
  • Boolean values must start with a capital letter: True or False
  • Strings can store numbers but they cannot be used in calculations
# Python automatically determines data types x = 5 # int y = 3.14 # float name = "Alice" # str is_student = True # bool (capital T!)
‚ö†Ô∏è Common Exam Question: When declaring variables in Python, do you need to specify a data type? NO!

üîÑ Type Conversion Functions

Function Purpose Example
str() Convert to string (prevents calculations) serialnumber = str(55555)
float() Convert to decimal number amount = float(44)
int() Convert to whole number age = int("25")
Practice: A serial number should not be used in calculations, and a dollar amount needs decimals.

üõ°Ô∏è Try/Except Blocks

x = float(input("Enter a number: ")) y = float(input("Enter a number to divide by: ")) try: print(f"The answer is {x/y}.") except: if y == 0: raise Exception("You cannot divide by zero") finally: print("Thank you for playing.")
Error Handling Keywords:
  • try: - Attempt to run code that might fail
  • except: - Code to run if error occurs
  • raise - Throw an exception
  • finally: - Runs no matter what (error or not)

13. Common Exam Mistakes to Avoid

‚ö†Ô∏è Top Mistakes Students Make

1. Order of Operations

2. List Methods
# WRONG - lists don't have .add() animals.add("Jaguars") # RIGHT - use .append() animals.append("Jaguars")
3. Boolean Capitalization
# WRONG - lowercase is_student = true # RIGHT - capital T and F is_student = True is_teacher = False
4. Calling Functions
# WRONG - don't use 'def' or 'call' order_total = def subtotal(500, 0.07) order_total = call subtotal(500, 0.07) # RIGHT - just use function name order_total = subtotal(500, 0.07)
5. Range Function
# range(1, 6) gives: 1, 2, 3, 4, 5 (NOT 6!) for i in range(1, 6): print(i) # Prints 1 through 5
