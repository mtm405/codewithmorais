The Creative App Challenge

üéØ Project Goal

Design, develop, and present a functional and creative application using Python. This project is your chance to apply all concepts learned (data types, lists, loops, functions, I/O, error handling) to solve a small problem, automate a task, or build an interactive program.

Minimum Code Length: Your final program file(s) must contain a minimum of 30 executable lines (excluding comments, blank lines, and imports).

Code Quality: Comments are mandatory. Every major section of logic (e.g., function, loop, complex calculation) must have a concise, descriptive comment explaining what it does.

üí° Creative Freedom & Advanced Theme Ideas

Your app's theme is entirely up to you, but it should be a bit more challenging than a simple, single-operation calculator.

Data & Tracking

Personalized Study Tracker

Key Feature: Track subjects, assign a score (float), calculate the weighted average (using a List), and suggest which subject to focus on next (using if/elif/else).

Text & Logic

Simple Password Validator

Key Feature: Accept a password (string) and use a for loop to check if it meets multiple criteria (length, contains a number, contains a symbol). Return a "Strength" score.

Interactive Game

Number Guessing Game (Hard Mode)

Key Feature: The computer picks a number (using the random module). Use a while loop to let the user guess until they get it right. Use a List to track all previous guesses.

Simulation

Basic Loan/Savings Simulator

Key Feature: Accept principal, interest rate, and term (years). Use a for loop to calculate the balance after each year, displaying the total interest paid.
