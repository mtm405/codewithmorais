1Ô∏è‚É£ The File System & Command-Line Toolkit

Objective 6.1: Master the modules that help your code interact with the computer's operating system

These modules help your code interact with the computer itself‚Äîthe files, folders, and the commands you type into the terminal. We call this the Operating System (OS) level.

Module What It Is Analogy Key Jobs (Exam Focus)
import os Operating System The Manager Creating folders, seeing your current location, changing directories
import os.path OS Path Utilities The GPS Checking if a file exists. Joining folder names and file names correctly
import sys System Interaction The Messenger Reading command-line arguments (input you give when running the program)
import io Input/Output The Librarian Opening and reading file contents (like text)

üìÅ The os Module: Your File System Manager

The os module is your gateway to interacting with the operating system. It allows you to navigate folders, create directories, check your current location, and much more.

üíª Common os Module Functions

import os # Get current working directory current_dir = os.getcwd() print(f"I'm currently in: {current_dir}") # Create a new directory os.mkdir("my_new_folder") # Change directory os.chdir("my_new_folder") # List files in current directory files = os.listdir() print(f"Files here: {files}") # Remove a directory os.rmdir("old_folder")

üí° Pro Tip: getcwd() vs chdir()

Think of getcwd() as asking "Where am I?" and chdir() as saying "Take me to this location." They're like GPS coordinates for your program!

üó∫Ô∏è The os.path Module: Your Navigation GPS

The os.path module is all about paths‚Äîthe addresses of files and folders on your computer. It helps you build paths correctly and check if files exist.

üõ£Ô∏è Path Navigation Examples

import os.path # Check if a file exists if os.path.exists("data.txt"): print("File found!") else: print("File not found!") # Join folder and file names correctly file_path = os.path.join("documents", "reports", "sales.txt") print(file_path) # Works on Windows, Mac, and Linux! # Check if it's a file or directory print(os.path.isfile("data.txt")) # True if it's a file print(os.path.isdir("documents")) # True if it's a directory # Get file size in bytes size = os.path.getsize("data.txt") print(f"File size: {size} bytes")

‚ö†Ô∏è Path Separators Matter!

Windows uses backslashes (\), while Mac/Linux use forward slashes (/). Always use os.path.join() to build paths‚Äîit automatically uses the right separator for your operating system!

üì® The sys Module: Your System Messenger

The sys module handles system-specific parameters and functions. Its most important feature is reading command-line arguments.

üí° Key Concept: Command-Line Arguments

When you run a Python script, you can pass extra information to it. These are arguments.

python my_script.py sales_data.txt 2024

The sys module is what lets your script "read" those extra pieces (sales_data.txt and 2024). It's how you make your scripts flexible!

üéØ Reading Command-Line Arguments

import sys # sys.argv is a list of all command-line arguments # sys.argv[0] is always the script name # sys.argv[1], sys.argv[2], etc. are your arguments print("Script name:", sys.argv[0]) if len(sys.argv) > 1: filename = sys.argv[1] print(f"Processing file: {filename}") else: print("No filename provided!") # Example: python my_script.py data.txt # Output: Processing file: data.txt

üí° Real-World Use Case

Imagine you have a script that processes different data files. Instead of hardcoding the filename, you can pass it as a command-line argument. This makes your script reusable for any file!

üìö The io Module: Your File Librarian

The io module provides Python's main facilities for dealing with various types of I/O (Input/Output). While you often use built-in functions like open(), the io module gives you more control over file operations.

üìñ File Reading Example

import io # Open and read a text file with io.open("data.txt", "r", encoding="utf-8") as file: content = file.read() print(content) # Read line by line with io.open("data.txt", "r") as file: for line in file: print(line.strip()) # Remove newline characters

üí° The "with" Statement

Always use with when opening files! It automatically closes the file when you're done, even if an error occurs. This prevents memory leaks and file corruption.

üî¢ A. The Math Module: Your Scientific Calculator

The math module is your high-powered scientific calculator. It provides access to mathematical functions and constants that are essential for any serious computation.

Function What It Does Example Use
math.fabs(x) Returns the absolute value of x math.fabs(-5) returns 5.0
math.ceil(x) Rounds x up to the nearest whole number math.ceil(4.1) returns 5
math.floor(x) Rounds x down to the nearest whole number math.floor(4.9) returns 4
math.trunc(x) Cuts off the decimal part (truncates) math.trunc(4.99) returns 4
math.sqrt(x) Calculates the square root math.sqrt(25) returns 5.0
math.pi A constant for the value of œÄ (‚âà3.14159...) Used for circle calculations
math.nan Represents "Not a Number" Used for handling errors in calculations

üßÆ Math Module in Action

import math # Absolute value distance = math.fabs(-42.5) print(f"Distance: {distance}") # 42.5 # Rounding operations price = 19.99 ceiling = math.ceil(price) # 20 floor = math.floor(price) # 19 truncated = math.trunc(price) # 19 print(f"Ceiling: {ceiling}, Floor: {floor}, Truncated: {truncated}") # Circle calculations radius = 5 circumference = 2 * math.pi * radius area = math.pi * (radius ** 2) print(f"Circumference: {circumference:.2f}") print(f"Area: {area:.2f}") # Square root side_length = math.sqrt(144) print(f"Side length: {side_length}") # 12.0

üí° Ceiling vs Floor vs Truncate

Ceiling: Always rounds UP (4.1 ‚Üí 5, -4.9 ‚Üí -4)
Floor: Always rounds DOWN (4.9 ‚Üí 4, -4.1 ‚Üí -5)
Truncate: Just removes decimals (4.9 ‚Üí 4, -4.9 ‚Üí -4)

üìÖ B. The Datetime Module: Your Time & Date Expert

The datetime module makes working with dates and times much easier than doing it manually. It's essential for scheduling, logging, and any time-based operations.

Function What It Does Analogy
datetime.now() Grabs the current time and date right now The Clock-In Stamp
date_obj.weekday() Tells you what day of the week (Monday is 0) The Day Planner
date_obj.strftime(format) Formats the date/time into a readable string The Translator

‚è∞ Datetime Module in Action

import datetime # Get current date and time now = datetime.datetime.now() print(f"Current datetime: {now}") # Get just the date today = datetime.date.today() print(f"Today's date: {today}") # What day of the week is it? (0=Monday, 6=Sunday) day_number = today.weekday() days = ["Monday", "Tuesday", "Wednesday", "Thursday", "Friday", "Saturday", "Sunday"] print(f"Today is {days[day_number]}") # Format dates in different ways formatted1 = now.strftime("%B %d, %Y") # December 25, 2025 formatted2 = now.strftime("%m/%d/%Y") # 12/25/2025 formatted3 = now.strftime("%I:%M %p") # 03:45 PM print(f"Format 1: {formatted1}") print(f"Format 2: {formatted2}") print(f"Format 3: {formatted3}") # Create a specific date birthday = datetime.date(2025, 12, 25) print(f"Birthday: {birthday}") # Calculate time difference days_until = (birthday - today).days print(f"Days until birthday: {days_until}")

üí° strftime Format Codes

%Y = 4-digit year (2025)
%m = Month as number (01-12)
%d = Day of month (01-31)
%B = Full month name (December)
%A = Full day name (Monday)
%I = Hour 12-hour format
%M = Minute (00-59)
%p = AM/PM

üé≤ C. The Random Module: Your Chance Generator

The random module is your source for chance and unpredictability. It's crucial for games, simulations, and anything that needs to be non-deterministic (not the same every time).

Function What It Does Example Use
random.randint(a, b) Returns a random integer between a and b (inclusive) random.randint(1, 6) for rolling a die
random.choice(list) Picks a single random item from a list random.choice(["Red", "Blue", "Green"])
random.shuffle(list) Mixes up (shuffles) the order of items in a list Shuffling a deck of cards
random.sample(list, k) Returns a list of k unique random items Picking 3 winners from a list of contestants

üé∞ Random Module in Action

import random # Roll a die (1-6) die_roll = random.randint(1, 6) print(f"You rolled a {die_roll}!") # Pick a random color colors = ["Red", "Blue", "Green", "Yellow", "Purple"] chosen_color = random.choice(colors) print(f"Random color: {chosen_color}") # Shuffle a deck of cards deck = ["A‚ô†", "K‚ô†", "Q‚ô†", "J‚ô†", "10‚ô†"] print(f"Original deck: {deck}") random.shuffle(deck) print(f"Shuffled deck: {deck}") # Pick 3 random winners (no duplicates) contestants = ["Alice", "Bob", "Charlie", "David", "Eve"] winners = random.sample(contestants, 3) print(f"Winners: {winners}") # Generate a random password characters = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789" password = "".join(random.choices(characters, k=12)) print(f"Random password: {password}")

‚ö†Ô∏è shuffle() vs sample()

shuffle(): Modifies the original list in place. The list order changes permanently.
sample(): Returns a NEW list with random selections. The original list stays unchanged.

3Ô∏è‚É£ The Most Important Step: Importing

None of these tools are available to your code until you explicitly ask for them using the import keyword.

Method 1: Import Entire Module

Import the whole toolbox. You need to use the module name when calling functions.

import math result = math.sqrt(16) pi_value = math.pi

‚úÖ Use when: You need many functions from the module

Method 2: Import Specific Functions

Import only what you need. You can use functions directly without the module name.

from math import sqrt, pi result = sqrt(16) pi_value = pi

‚úÖ Use when: You only need a few specific functions

Method 3: Import with Alias

Give the module a shorter nickname to save typing.

import datetime as dt now = dt.datetime.now() today = dt.date.today()

‚úÖ Use when: The module name is long

Method 4: Import Everything (‚ö†Ô∏è Not Recommended)

Import all functions from a module. Can cause naming conflicts!

from math import * result = sqrt(16) pi_value = pi

‚ö†Ô∏è Avoid: Can overwrite existing functions and make code confusing

üéØ Best Practice Example

# Organize all imports at the TOP of your file import os import sys import math from datetime import datetime, date from random import randint, choice # Now you can use all these tools in your code print("Current directory:", os.getcwd()) print("Today:", date.today()) print("Random number:", randint(1, 100)) print("Square root of 64:", math.sqrt(64))
