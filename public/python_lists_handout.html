Python Lists: Complete Handout

A comprehensive reference guide for mastering lists in Python

1. Creating and Understanding Lists

What is a List?

A list is an ordered, mutable collection that can hold multiple items of any data type.

Key Characteristics: Ordered (indexed), Mutable (changeable), Flexible (any type), Indexed (0-based)

Creating Lists

fruits = ["apple", "banana", "orange"] numbers = [1, 2, 3, 4, 5] mixed = ["Hello", 42, 3.14, True] empty = []

2. Accessing Items: Indexing

Positive Indexing (left to right)

CodeValue
fruits[0]"apple"
fruits[1]"banana"
fruits[2]"orange"

Negative Indexing (right to left)

CodeValue
fruits[-1]"orange" (last)
fruits[-2]"banana" (second-to-last)
fruits[-3]"apple" (third-to-last)

3. Slicing Lists

Slice Syntax: list[start:stop:step]

  • start: Index to begin (included)
  • stop: Index to end (NOT included)
  • step: Gap between items (default 1)

Slice Examples

numbers = [0, 1, 2, 3, 4, 5] numbers[1:4] # [1, 2, 3] numbers[:3] # [0, 1, 2] numbers[2:] # [2, 3, 4, 5] numbers[::2] # [0, 2, 4] numbers[::-1] # [5, 4, 3, 2, 1, 0]

List Methods & Techniques

4. Adding and Removing Items

append(item) - Add one item to end

fruits = ["apple", "banana"] fruits.append("orange") # fruits = ["apple", "banana", "orange"]

extend(items) - Add multiple items

fruits = ["apple", "banana"] fruits.extend(["orange", "grape"]) # fruits = ["apple", "banana", "orange", "grape"]

insert(index, item) - Add at specific position

letters = ["a", "c", "d"] letters.insert(1, "b") # letters = ["a", "b", "c", "d"]

remove(item) - Remove first matching item

colors = ["red", "blue", "red"] colors.remove("red") # Removes first "red" # colors = ["blue", "red"]

pop(index) - Remove and return item

numbers = [1, 2, 3, 4] last = numbers.pop() # Returns 4, list = [1, 2, 3] first = numbers.pop(0) # Returns 1, list = [2, 3]

clear() - Remove all items

colors = ["red", "blue"] colors.clear() # colors = []

5. Finding and Sorting

index(item) - Find position

fruits = ["apple", "banana", "orange"] position = fruits.index("banana") # Returns 1

count(item) - Count occurrences

numbers = [1, 2, 2, 3, 2] count = numbers.count(2) # Returns 3

sort() - Arrange in order

numbers = [3, 1, 4, 1, 5] numbers.sort() # numbers = [1, 1, 3, 4, 5]

reverse() - Reverse order

numbers = [1, 2, 3] numbers.reverse() # numbers = [3, 2, 1]

6. Advanced Techniques

Nested Lists (lists within lists)

matrix = [[1, 2], [3, 4], [5, 6]] matrix[0] # [1, 2] matrix[0][1] # 2 matrix[1][0] # 3

List Comprehension

# Create squares numbers = [1, 2, 3, 4] squared = [x*x for x in numbers] # [1, 4, 9, 16] # Filter even numbers evens = [x for x in numbers if x % 2 == 0] # [2, 4]
Remember: Most methods modify the original list! Only index(), count(), and slices return new lists without changing the original.
