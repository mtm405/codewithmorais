What Are Lists?

Definition: Lists are ordered, mutable collections that can hold multiple items of different data types.

Key Characteristics:

  • Ordered: Items have a position (index)
  • Mutable: You can change, add, or remove items
  • Flexible: Can contain strings, numbers, other lists, etc.
  • Indexed: Access items by position (starting from 0)

üîçUnderstanding Indexing

Positive Indexing (left to right):

  • Index 0 = First item
  • Index 1 = Second item
  • Index 2 = Third item

Negative Indexing (right to left):

  • Index -1 = Last item
  • Index -2 = Second-to-last item
  • Index -3 = Third-to-last item

üìñIndexing: Examples

fruits = ["apple", "banana", "orange"]
# 0 1 2 (positive)
# -3 -2 -1 (negative)
# Positive indexing
fruits[0] # "apple" (first)
fruits[1] # "banana" (second)
fruits[2] # "orange" (third)
# Negative indexing
fruits[-1] # "orange" (last)
fruits[-2] # "banana" (second-to-last)
fruits[-3] # "apple" (third-to-last)

üîçUnderstanding Slicing

Slice Syntax: list[start:stop:step]

  • start: Where to begin (included)
  • stop: Where to end (NOT included)
  • step: Gap between items (default is 1)

Common Slice Patterns:

list[1:4] - Items at index 1, 2, 3
list[:3] - First 3 items (index 0, 1, 2)
list[2:] - From index 2 to the end
list[::2] - Every 2nd item (skip one, take one)

üìñSlicing: Examples

numbers = [0, 1, 2, 3, 4, 5]
# Basic slicing
numbers[1:4] # [1, 2, 3]
numbers[:3] # [0, 1, 2]
numbers[2:] # [2, 3, 4, 5]
# Step parameter
numbers[::2] # [0, 2, 4] (every 2nd)
numbers[1::2] # [1, 3, 5]
numbers[::-1] # [5, 4, 3, 2, 1, 0] (reversed)

üîçUnderstanding append()

What it does:

The append() method adds ONE item to the END of a list.

Important Notes:

  • Only takes ONE argument (the item to add)
  • Modifies the original list (doesn't return anything)
  • Adds to the end, not anywhere in the middle

üìñappend(): Examples

numbers = [1, 2, 3]
numbers.append(4)
print(numbers) # [1, 2, 3, 4]
numbers.append(5)
print(numbers) # [1, 2, 3, 4, 5]
# Append different types
fruits = ["apple"]
fruits.append("banana")
fruits.append(42)
print(fruits) # ["apple", "banana", 42]

‚öñÔ∏èextend() vs append()

# append() - Adds ONE thing (even if it's a list) fruits = ["apple", "banana"] fruits.append(["orange", "grape"]) print(fruits) # ["apple", "banana", ["orange", "grape"]]
# extend() - Adds MULTIPLE items separately fruits = ["apple", "banana"] fruits.extend(["orange", "grape"]) print(fruits) # ["apple", "banana", "orange", "grape"]

üîçUnderstanding pop()

What it does:

The pop() method removes an item AND returns it.

Key differences from remove():

  • pop(): Uses index, returns the item removed
  • remove(): Uses value, returns nothing

üìñpop(): Examples

numbers = [10, 20, 30, 40]
# Pop without index (removes last)
last = numbers.pop()
print(last) # 40
print(numbers) # [10, 20, 30]
# Pop with index (removes at position)
first = numbers.pop(0)
print(first) # 10
print(numbers) # [20, 30]

üîéindex() & count()

index(item) - Find position:

count(item) - Count occurrences:

numbers = [1, 2, 2, 3, 2] count = numbers.count(2) print(count) # 3

‚ÜïÔ∏èsort() & reverse()

sort() - Arrange in order:

numbers = [3, 1, 4, 1, 5] numbers.sort() print(numbers) # [1, 1, 3, 4, 5]

reverse() - Reverse the order:

numbers = [1, 2, 3, 4, 5] numbers.reverse() print(numbers) # [5, 4, 3, 2, 1]

üîçUnderstanding Nested Lists

What are nested lists?

Lists that contain other lists as elements.

Accessing nested items:

matrix = [[1, 2], [3, 4], [5, 6]] matrix[0] # [1, 2] (first row) matrix[0][0] # 1 (first item of first row) matrix[0][1] # 2 (second item of first row) matrix[1][0] # 3 (first item of second row)

üîçUnderstanding List Comprehension

What is it?

A concise way to create a new list by transforming or filtering an existing list.

Basic Syntax:

[expression for item in iterable]

With condition:

[expression for item in iterable if condition]

üìñList Comprehension: Examples

# Create squares
squared = [x*x for x in [1, 2, 3, 4]]
# [1, 4, 9, 16]
# Filter even numbers
evens = [x for x in [1, 2, 3, 4] if x % 2 == 0]
# [2, 4]
# Create uppercase words
words = ["hello", "world"]
upper = [word.upper() for word in words]
# ["HELLO", "WORLD"]
