üîî Bell Ringer: The Lost Data Problem (5 Minutes)

Question:

A student wrote this code to save their high score. But every time they run it, their old score disappears!

# File has: "Score: 500"

score = 650
with open('game.txt', 'w') as f:
    f.write('Score: ' + str(score))

# Result: File now only has "Score: 650"
# The 500 is gone!

# What went wrong?
Click to see the answer

The Problem: Using 'w' mode erases everything in the file!

Why it happens: The 'w' mode means "write from scratch" - it deletes the old content first.

The Fix: Use 'a' mode to add to the end without erasing: 

with open('game.txt', 'a') as f:
    f.write('Score: ' + str(score) + '\n')

üìù Part 1: The Three File Modes

When you open a file, you need to tell Python what you want to do. There are three simple modes:

Mode What It Does When to Use It
'w' Write - Erases the file and starts fresh Creating a new file or completely replacing old content
'a' Append - Adds to the end of the file Adding new data without losing old data
'r' Read - Look at what's in the file Reading data from a file

Simple Examples:

Example 1: Write Mode - Starting Fresh

# Create a new file with a header
with open('scores.txt', 'w') as f:
    f.write('Game High Scores\n')
    f.write('================\n')

Example 2: Append Mode - Adding More

# Add a new score to the file
with open('scores.txt', 'a') as f:
    f.write('Player: Alice - Score: 1500\n')
    f.write('Player: Bob - Score: 1200\n')

Example 3: Read Mode - Looking at Data

# Read everything in the file
with open('scores.txt', 'r') as f:
    content = f.read()
    print(content)
üí° Remember: Always use with open() - it automatically closes the file when done!

üìñ Part 2: Reading Files Different Ways

There are three ways to read data from a file:

Method What It Does Example Result
f.read() Reads the entire file as one string "Line 1\nLine 2\nLine 3"
f.readline() Reads just one line "Line 1\n"
f.readlines() Reads all lines into a list ["Line 1\n", "Line 2\n", "Line 3"]

Complete Example:

# Create a file with data
with open('names.txt', 'w') as f:
    f.write('Alice\n')
    f.write('Bob\n')
    f.write('Charlie\n')

# Method 1: Read everything at once
with open('names.txt', 'r') as f:
    all_text = f.read()
    print("All at once:")
    print(all_text)

# Method 2: Read line by line
with open('names.txt', 'r') as f:
    line1 = f.readline()
    line2 = f.readline()
    print("\nOne at a time:")
    print("First:", line1.strip())
    print("Second:", line2.strip())

# Method 3: Read as a list
with open('names.txt', 'r') as f:
    all_lines = f.readlines()
    print("\nAs a list:")
    for name in all_lines:
        print("-", name.strip())
üí° Pro Tip: Use .strip() to remove the \n at the end of each line!

üî¢ Part 3: Formatting Numbers in Files

When saving numbers (especially money), you need to format them properly:

The Problem:

price = 12.5
print(price)  # Shows: 12.5
# But we want: 12.50 (two decimal places)

The Solution: Use :.2f

price = 12.5
formatted = f"{price:.2f}"
print(formatted)  # Shows: 12.50

# Another example
price = 99.999
formatted = f"{price:.2f}"
print(formatted)  # Shows: 100.00 (rounds up!)

Real Example: Saving Prices to a File

# Save prices with proper formatting
with open('prices.txt', 'w') as f:
    f.write('Product Prices\n')
    f.write('==============\n')
    
    # Write each price with 2 decimal places
    apple_price = 1.5
    f.write(f'Apples: ${apple_price:.2f}\n')
    
    banana_price = 0.75
    f.write(f'Bananas: ${banana_price:.2f}\n')
    
    orange_price = 2.0
    f.write(f'Oranges: ${orange_price:.2f}\n')

# File will contain:
# Product Prices
# ==============
# Apples: $1.50
# Bananas: $0.75
# Oranges: $2.00
Important: The :.2f means "format as a decimal number with exactly 2 digits after the decimal point"

üìù Part 4: Working with Lists and Files

You can combine lists with file operations to save and load data easily!

Saving a List to a File:

# Save student names to a file
students = ['Alice', 'Bob', 'Charlie', 'Diana']

with open('students.txt', 'w') as f:
    for student in students:
        f.write(student + '\n')

print("Students saved!")

Loading a List from a File:

# Read student names back into a list
students = []

with open('students.txt', 'r') as f:
    lines = f.readlines()
    for line in lines:
        students.append(line.strip())

print("Students loaded:", students)
# Output: Students loaded: ['Alice', 'Bob', 'Charlie', 'Diana']

Shorter Way (Advanced Tip):

# You can loop directly through the file!
students = []

with open('students.txt', 'r') as f:
    for line in f:
        students.append(line.strip())

print(students)
üí° Cool Trick: You can loop through a file object directly - each iteration gives you one line!

‚ö†Ô∏è Part 6: Common Mistakes to Avoid

Mistake #1: Forgetting the Newline

# ‚ùå WRONG - Everything on one line!
with open('data.txt', 'w') as f:
    f.write('Line 1')
    f.write('Line 2')
    f.write('Line 3')
# File contains: Line 1Line 2Line 3

# ‚úÖ CORRECT - Add \n for new lines
with open('data.txt', 'w') as f:
    f.write('Line 1\n')
    f.write('Line 2\n')
    f.write('Line 3\n')
# File contains:
# Line 1
# Line 2
# Line 3

Mistake #2: Using 'w' Instead of 'a'

# ‚ùå WRONG - Erases old data every time!
with open('log.txt', 'w') as f:
    f.write('New entry\n')
# Old entries are GONE!

# ‚úÖ CORRECT - Adds to the end
with open('log.txt', 'a') as f:
    f.write('New entry\n')
# Old entries are kept!

Mistake #3: Not Closing Files (Fixed by 'with')

# ‚ùå OLD WAY - Must remember to close!
f = open('data.txt', 'w')
f.write('Hello')
f.close()  # Easy to forget!

# ‚úÖ BEST WAY - Automatically closes
with open('data.txt', 'w') as f:
    f.write('Hello')
# File closes automatically!

Mistake #4: Forgetting to Convert Numbers

# ‚ùå WRONG - Can't write numbers directly
score = 100
with open('score.txt', 'w') as f:
    f.write(score)  # ERROR!

# ‚úÖ CORRECT - Convert to string
score = 100
with open('score.txt', 'w') as f:
    f.write(str(score))  # Works!
    # Or use f-string:
    f.write(f'Score: {score}\n')  # Even better!

üìã Quick Reference Card

File Modes:

  • 'w' = Write (erases old content)
  • 'a' = Append (adds to end)
  • 'r' = Read (just look, don't change)

Reading Methods:

  • f.read() = Read entire file
  • f.readline() = Read one line
  • f.readlines() = Read all lines as a list

Important Tips:

  • Always use with open() to open files
  • Add \n at the end of each line when writing
  • Use .strip() to remove \n when reading
  • Format numbers with {number:.2f} for 2 decimal places
  • Convert numbers to strings before writing: str(number)
  • You can loop directly through a file object
The Golden Rule: Think before you open! Are you creating something new ('w'), adding to existing ('a'), or just reading ('r')?

# Daily habit data (True = completed, False = not done)
habits = {
    'Exercise': [True, True, False, True, True, True, False],
    'Reading': [True, True, True, True, False, True, True],
    'Coding': [True, False, True, True, True, False, True]
}

days = ['Mon', 'Tue', 'Wed', 'Thu', 'Fri', 'Sat', 'Sun']

# Save to file
with open('habits.txt', 'w') as f:
    f.write('üìä Weekly Habit Tracker\n')
    f.write('=' * 45 + '\n\n')
    
    # Header
    f.write('Habit        ')
    for day in days:
        f.write(f'{day}  ')
    f.write('Rate\n')
    f.write('-' * 45 + '\n')
    
    # Each habit
    for habit, completed in habits.items():
        f.write(f'{habit:12} ')
        for done in completed:
            symbol = '‚úì' if done else '‚úó'
            f.write(f'{symbol}   ')
        
        # Calculate percentage
        percentage = (sum(completed) / len(completed)) * 100
        f.write(f' {percentage:.0f}%\n')

# Display file
with open('habits.txt', 'r') as f:
    print(f.read())

# Calculate overall statistics
total_tasks = 0
completed_tasks = 0

for completed in habits.values():
    total_tasks += len(completed)
    completed_tasks += sum(completed)

overall = (completed_tasks / total_tasks) * 100
print(f'\nüìà Overall completion: {overall:.1f}%')

  4. Reads the file back and calculates your average
  5. Displays the average with the message: "Your average is: XX.XX"

# Personal Score Card
name = "Your Name"
date = "November 13, 2025"

# Create scorecard
with open('scorecard.txt', 'w') as f:
    f.write(f'Score Card for {name}\n')
    f.write(f'Date: {date}\n')
    f.write('=' * 30 + '\n\n')

# Add scores
scores = {
    'Math': 88,
    'Science': 92,
    'English': 85,
    'History': 90
}

with open('scorecard.txt', 'a') as f:
    for subject, score in scores.items():
        f.write(f'{subject}: {score}\n')

# Calculate average
total = 0
count = 0

with open('scorecard.txt', 'r') as f:
    for line in f:
        if ':' in line and not 'Date' in line and not 'Card' in line:
            score = int(line.split(':')[1].strip())
            total += score
            count += 1

average = total / count

# Determine letter grade (BONUS)
if average >= 90:
    grade = 'A'
elif average >= 80:
    grade = 'B'
elif average >= 70:
    grade = 'C'
elif average >= 60:
    grade = 'D'
else:
    grade = 'F'

# Display results
print(f'\nYour average is: {average:.2f}')
print(f'Letter Grade: {grade}')

# Show the scorecard
print('\n--- Your Score Card ---')
with open('scorecard.txt', 'r') as f:
    print(f.read())
