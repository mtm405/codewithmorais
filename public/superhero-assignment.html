ü§î Troubleshooting

My program crashes with "No module named 'superhero'"

Solution: Make sure both files are in the same folder, and you've added import superhero to main.py.

I get "function returned None" or no output

Solution: Check that all your functions have return statements. Each function should send data back!

The superhero name isn't combining correctly

Solution: Make sure you're using spaces between the parts: " " (that's a space in quotes).
