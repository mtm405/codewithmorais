Project Overview

Learning Objectives

In this project, you will learn essential Microsoft Word 365 Online skills by creating a professional Thanksgiving dinner invitation. You'll explore formatting, design elements, and document organization while celebrating the holiday season!

Note: This project is designed for Word 365 Online. Access it at office.com and sign in with your Microsoft account.

What You'll Create

A beautifully designed, one-page Thanksgiving dinner invitation that includes:

  • Eye-catching header with festive graphics
  • Event details (date, time, location)
  • Menu highlights
  • RSVP information
  • Decorative elements and seasonal imagery

Step-by-Step Instructions

Part 1: Document Setup

  1. Create a New Document

    Go to office.com ‚Üí Sign in ‚Üí Click "Word" to create a new blank document. The document auto-saves to OneDrive, so name it "Thanksgiving_Invitation_YourName" by clicking the title at the top.

  2. Set Page Layout

    Click the Layout tab ‚Üí Click Margins ‚Üí Select "Narrow" (0.5" all around). This gives you more space for your design.

  3. Add Page Border

    Click the Design tab ‚Üí Click Page Borders (on the right side) ‚Üí Under "Setting" choose "Box" ‚Üí Click the "Art" dropdown and choose a decorative border (scroll to find fall/autumn themes like leaves) ‚Üí Set color to orange or brown ‚Üí Click OK.

Word 365 Online auto-saves your work to OneDrive automatically - no need to manually save!

Part 2: Create the Header

  1. Insert WordArt Title

    Click the Insert tab ‚Üí Click WordArt ‚Üí Choose a style (like "Fill: Orange, Accent color 2; Outline: Orange, Accent color 2"). Type "You're Invited!" or "Thanksgiving Celebration". The WordArt will appear in the document.

  2. Add Decorative Images

    Click the Insert tab ‚Üí Click Icons ‚Üí In the search box, type "thanksgiving", "turkey", "pumpkin", or "fall leaves" ‚Üí Select 2-3 icons you like ‚Üí Click Insert. Click and drag the corners to resize. Click and drag the image to position it around your title.

  3. Format Title Text

    Click on your WordArt ‚Üí The Shape Format tab appears ‚Üí Click Text Effects ‚Üí Choose Shadow or Glow ‚Üí Select a preset effect for extra visual appeal.

Part 3: Event Details

  1. Add Event Information

    Type the following details (press Enter after each line):

    • Join us for Thanksgiving Dinner!
    • Date: Thursday, November 28, 2025
    • Time: 3:00 PM - 8:00 PM
    • Location: 456 Autumn Lane, Harvest Hills, CA 95123
  2. Format Event Details

    Click and drag to select all event text ‚Üí In the Home tab, click the Font Size dropdown ‚Üí Choose 14 or 16 ‚Üí Click the Center Align button. Select the labels (Date:, Time:, Location:) ‚Üí Click Bold button ‚Üí Click the Font Color dropdown to choose a darker color.

  3. Add a Text Box

    Click the Insert tab ‚Üí Click Text Box ‚Üí Click Draw Text Box ‚Üí Click and drag to draw the box. Type your message like: "Bring your appetite and gratitude!" ‚Üí Click the border of the text box ‚Üí The Shape Format tab appears ‚Üí Click Shape Fill to add color ‚Üí Click Shape Outline to change border.

Part 4: Menu Highlights

  1. Create Section Header

    Type "Menu Highlights" ‚Üí Select the text ‚Üí In the Home tab, set font size to 18pt, click Bold, and choose an autumn color from the Font Color dropdown. To add a line below: Click Insert tab ‚Üí Shapes ‚Üí Click the straight line ‚Üí Click and drag below your header.

  2. Insert a Table

    Click the Insert tab ‚Üí Click Table ‚Üí In the grid, select 2 columns and 4 rows (move your mouse to highlight 2x4 cells) ‚Üí Click to insert. Type the following in each cell:

    • Row 1: Roasted Turkey | Glazed Ham
    • Row 2: Mashed Potatoes | Sweet Potato Casserole
    • Row 3: Green Bean Casserole | Cranberry Sauce
    • Row 4: Pumpkin Pie | Apple Pie
  3. Format the Table

    Click anywhere in the table ‚Üí The Table Design tab appears ‚Üí Click on different table styles in the gallery to preview ‚Üí Choose one with autumn colors (orange/brown tones). For alignment: Click Layout tab ‚Üí Select cells ‚Üí Click Align Center.

To match colors in your document, note the color names when you select them (like "Orange, Accent 2") and use the same color name throughout for consistency.

Part 5: RSVP Information

  1. Add RSVP Section

    Type "RSVP" ‚Üí Select it ‚Üí Set font size to 20pt and click Bold. Press Enter and type:

    • Please respond by: November 20, 2025
    • Contact: Sarah Thompson
    • Phone: (555) 123-4567
    • Email: thanksgiving2025@email.com
  2. Add Decorative Shape

    Click Insert tab ‚Üí Shapes ‚Üí Choose Rectangle or Rounded Rectangle ‚Üí Click and drag to draw around your RSVP section ‚Üí Click the shape's border ‚Üí In Shape Format tab, click Shape Fill to add color ‚Üí Click Shape Outline to change border ‚Üí Right-click the shape ‚Üí Select Send to Back so text appears on top.

Part 6: Final Touches

  1. Add Footer Message

    At the bottom of your page, type: "We're thankful for you!" ‚Üí Select the text ‚Üí Click Italic button ‚Üí Change the font to something decorative like "Brush Script MT" or "Lucida Handwriting" from the Font dropdown.

  2. Insert More Graphics

    Click Insert tab ‚Üí Icons ‚Üí Search for "acorn", "corn", or "pumpkin" ‚Üí Insert small icons throughout the page. Resize them small and position in empty spaces to enhance the design.

  3. Review and Adjust

    Scroll through your document and check spacing, alignment, and overall balance. Click and drag elements to adjust positions. Make sure all text is readable and images don't overlap text.

  4. Spell Check

    Click the Review tab ‚Üí Click Check Document (or Spelling & Grammar) ‚Üí Review and fix any errors found. Word 365 Online also shows squiggly underlines for errors as you type.
